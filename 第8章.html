第8章 积分进一步应用

Further Applications of Integration — 弧长 · 旋转曲面面积 · 静水压力 · 力矩与质心 · Pappus 定理

8.1 弧长

▼曲线弧长公式 必考页1

弧长 = 对「无穷小斜边」积分：L = ∫ √[1+(dy/dx)²] dx

Hook：给你一条弯弯曲曲的曲线——你怎么量它的「长度」？拿尺子贴着量？不行——微积分的方法是把曲线切成无数个几乎直的小段，每段用勾股定理算长度再求和。

Why：两点之间直线最短，但曲线不是直的。把曲线放大到无穷小尺度，每一段近似为斜边 Δs ≈ √[(Δx)²+(Δy)²]，取极限就是积分。

What：对 y = f(x)，a ≤ x ≤ b：

$$L = \int_a^b \sqrt{1 + [f'(x)]^2}\,dx$$

若曲线用参数形式给出 x = x(t), y = y(t)（α ≤ t ≤ β）：

$$L = \int_\alpha^\beta \sqrt{[x'(t)]^2 + [y'(t)]^2}\,dt$$

How：求 f'(x) → 算 1+(f')² → 看能否化简为完全平方（常见技巧: 1+tan²=sec², (a+b)²展开等）→ 积分。

易错：弧长 > 割线长度。区间内任一小区间的弧长 ≥ √[(Δx)²+(Δy)²]；等号仅在 f'(x) 恒为常数（曲线为直线）时成立。考试陷阱：求「弧长」和求「弧长函数的导数」是两回事——弧长函数的导数 s'(x) = √[1+(f')²]。

对 y = x^(3/2) 从 (0,0) 到 (4,8)，弧长公式的被积函数是什么？

参考答案

y' = (3/2)x^(1/2)，1+(y')² = 1 + (9/4)x。被积函数 = √(1+9x/4)。

▼弧长函数 重点页1

Hook：如果以曲线上某个固定点为起点，走到曲线上任意一点的路程怎么算？

定义弧长函数 s(x) = ∫_a^x √[1+(f'(t))²] dt。它的导数 s'(x) = √[1+(f'(x))²] 就是弧长元素 ds/dx。也可以用 y 做自变量：s(y) = ∫_c^y √[1+(g'(t))²] dt。

弧长函数的导数 s'(x) 的几何意义是什么？

参考答案

s'(x) 表示曲线上 x 每增加一个单位，弧长增加的量——即「弧长微元 ds 与 dx 之比」。几何上等于该点处割线斜边与水平边之比（无穷小尺度下的勾股弦比）。

8.2 旋转曲面面积页2

▼绕坐标轴旋转的曲面面积 重点

S = ∫ 2π × (半径) × (弧长微元)

Hook：把一条曲线绕轴旋转——扫出来的曲面的面积是多少？想象把曲线切成无数小段，每小段绕轴转出一个「窄环带」，所有环带面积加起来。

Why：每个小弧段旋转一周扫出的面积近似为「周长 × 带宽」。周长 = 2π × (到转轴的距离)，带宽 = 弧长微元 ds。

What：绕 x 轴（y = f(x) ≥ 0）：

$$S = \int_a^b 2\pi\,y\,\sqrt{1+[f'(x)]^2}\,dx$$

绕 y 轴：用 x = g(y) 形式，S = ∫ 2πx·√[1+(dx/dy)²] dy。

易错：弧长公式和旋转面积公式长得很像——关键区别在于旋转面积多了一个 2π×(到轴的距离)。考试陷阱：绕 x 轴时半径是 y，绕 y 轴时半径是 x，用反了直接零分。

y = x² (0 ≤ x ≤ 2) 绕 x 轴旋转，曲面面积的被积函数是？设积分但不计算。

参考答案

S = ∫_0^2 2π·x²·√(1+4x²) dx。其中 2πx² 是周长因子(半径=y=x³)，√(1+9x⁴) 是 ds/dx。

8.3 物理与工程应用

▼静水压力与力 重点页2

F = ∫ ρg × (深度) × (微元面积) = 压力 × 面积 的积分

Hook：水坝为什么越往下越厚？因为水压随深度线性增长——底部承受的压力是顶部的几十倍。

Why：压强 p = ρgh（ρ 为密度，g 为重力加速度，h 为水深）。同一深度处压强处处相等（帕斯卡原理），但不同深度压强不同，所以总力需要对深度积分。

How：1) 建立坐标系（通常 y 轴向下，原点在水面）；2) 写出深度 d(y)；3) 写出该深度的微元宽度 w(y)；4) dF = ρg·d(y)·w(y)·dy；5) 积分。

易错：坐标系方向决定积分上下限——y 轴向下 vs 向上时深度表达式不同。另一个易错点：水坝形状是梯形时，宽度 w(y) 随深度线性变化，需要正确写出 w(y) 的线性函数。

▼力矩与质心 必考

质心是「平衡点」：x̄ = M_y / m = (∫ x dm) / (∫ dm)

Hook：一张不规则形状的薄板，用手指顶在哪一点能让它平衡不掉？这个点就是质心。

What：

概念离散点质量连续薄板（y=f(x)）
质量 mΣ m_k∫ ρ(x) f(x) dx
对 y 轴力矩 M_yΣ m_k x_k∫ x·ρ(x)·f(x) dx
对 x 轴力矩 M_xΣ m_k y_k∫ ½[f(x)]²·ρ(x) dx
质心 x̄M_y/mM_y/m
质心 ȳM_x/mM_x/m

形心（Centroid）是均匀密度薄板的质心：令 ρ=1 即可。形心的 ȳ = (1/A)∫ ½[f(x)]² dx。

易错：对 x 轴力矩的公式是 ∫ ½[f(x)]² dx，不是 ∫ f(x) dx——因为微元的「平均 y 坐标」是 f(x)/2，力矩 = (f(x)/2) × (f(x)dx) = ½[f(x)]²dx。这是最常错的公式。

由 y = √x、y = 0、x = 4 围成的区域的形心坐标是？

参考答案

A = ∫_0^4 √x dx = 16/3。x̄ = (3/16)∫_0^4 x√x dx = 12/5。ȳ = (3/16)∫_0^4 ½x dx = 3/4。形心 (12/5, 3/4)。

▼Pappus 定理 高频页3

Hook：一个圆绕外面的轴转一圈，得到一个甜甜圈——它的体积不用三重积分，一个公式秒出。

What：平面区域 R 绕不与之相交的轴旋转——旋转体体积 V = 2π·ȳ·A（其中 ȳ 是形心到转轴的距离）。平面曲线 C 绕轴旋转——旋转曲面面积 S = 2π·ȳ·L（其中 L 是曲线弧长，ȳ 是曲线形心到转轴的距离）。

半径为 r 的圆，圆心在 x = R（R > r），绕 y 轴旋转得到圆环体。用 Pappus 定理求其体积。

参考答案

圆的面积 A = πr²，圆的形心在圆心 x̄ = R。绕 y 轴旋转 → 形心走过的距离 = 2πR。由 Pappus 定理，V = (2πR)·(πr²) = 2π²Rr²。
