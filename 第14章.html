14.1  多元函数

▼定义页2

一个 二元函数 \(f:\mathbb{R}^2\to\mathbb{R}\) 给每一对有序实数 \((x,y)\in D\) 指定唯一实数 \(z=f(x,y)\)。 \(D\)＝定义域，值域＝\(\{f(x,y)\mid (x,y)\in D\}\)。

常见定义域类型（\(\mathbb{R}^2\) 子集）
  • 开区域：不含边界，如 \(\{(x,y)\mid x^2+y^2<1\}\)
  • 闭区域：含所有边界，如 \(\{(x,y)\mid x^2+y^2\le 1\}\)
  • 有界 v.s. 无界：是否存在一个圆盘包含整个集合

▼图像页1

\(z=f(x,y)\) 的图像是 \(\mathbb{R}^3\) 中的 曲面：\(\{(x,y,z)\mid z=f(x,y),\,(x,y)\in D\}\)。

▼等高线与等位面

等高线 (level curve)：对常数 \(k\)，曲线 \(f(x,y)=k\) 是曲面上高度为 \(k\) 的"横切面"投影到 \(xy\) 平面。
三元函数 \(w=f(x,y,z)\) 的 等位面：\(f(x,y,z)=k\)。

例：\(f(x,y)=x^2+y^2\) 的等高线是一族同心圆，越往外 \(z\) 越大 → 抛物面。

等高线密集的地方 → 曲面 陡；等高线稀疏的地方 → 曲面 缓。 这与地形图中"等高线密＝坡陡"完全一致。

  2. 画出 \(f(x,y)=x^2-y^2\) 的等高线草图（双曲抛物面／马鞍面）。

14.2  极限与连续性

▼二元极限定义

\(\displaystyle\lim_{(x,y)\to(a,b)} f(x,y)=L\) 的含义：无论 \((x,y)\) 沿任何路径逼近 \((a,b)\)，\(f(x,y)\) 都趋于同一值 \(L\)。

▼证明极限不存在 — 双路径法

策略：找 两条不同路径，沿它们逼近时得到 不同极限值。

常用路径：
  • 沿 \(x\) 轴：令 \(y=0\)，看 \(x\to 0\)
  • 沿 \(y\) 轴：令 \(x=0\)，看 \(y\to 0\)
  • 沿直线 \(y=mx\)
  • 沿抛物线 \(y=x^2\) 或 \(x=y^2\)

经典例题：证 \(\displaystyle\lim_{(x,y)\to(0,0)}\frac{xy}{x^2+y^2}\) 不存在。

  • 沿 \(x\) 轴 (\(y=0\))：极限 = 0
  • 沿 \(y=x\)：\(\frac{x^2}{2x^2}=\frac12\) ≠ 0 → 不存在
不能用"沿所有直线方向极限相同"来断言极限存在 —— 还必须考虑曲线路径！ 例：\(f(x,y)=\frac{x^2y}{x^4+y^2}\)，沿所有直线 \(y=mx\) 极限都是 0，但沿 \(y=x^2\) 极限 = 1/2 ≠ 0 → 极限不存在。

▼连续性页2

\(f\) 在 \((a,b)\) 连续 ⟺ \(\displaystyle\lim_{(x,y)\to(a,b)}f(x,y)=f(a,b)\)。
连续函数在有界闭区域上必有最大值和最小值（极值定理）。

14.3  偏导数

▼定义页2

\(\displaystyle f_x(a,b)=\lim_{h\to0}\frac{f(a+h,b)-f(a,b)}{h}\)
\(\displaystyle f_y(a,b)=\lim_{h\to0}\frac{f(a,b+h)-f(a,b)}{h}\)

\(f_x\)：将 \(y\) 视为常数，对 \(x\) 求导。
\(f_y\)：将 \(x\) 视为常数，对 \(y\) 求导。
记号：\(f_x \equiv \frac{\partial f}{\partial x} \equiv \frac{\partial z}{\partial x}\)。

▼二阶偏导数页3

记号含义
\(f_{xx}=\frac{\partial^2 f}{\partial x^2}\)先对 \(x\) 导再对 \(x\) 导
\(f_{yy}=\frac{\partial^2 f}{\partial y^2}\)先对 \(y\) 导再对 \(y\) 导
\(f_{xy}=\frac{\partial^2 f}{\partial y\partial x}\)先对 \(x\) 导，再对 \(y\) 导
\(f_{yx}=\frac{\partial^2 f}{\partial x\partial y}\)先对 \(y\) 导，再对 \(x\) 导

▼Clairaut 定理

若 \(f_{xy}\) 和 \(f_{yx}\) 都在 \((a,b)\) 连续，则
\(f_{xy}(a,b)=f_{yx}(a,b)\) 混合偏导的次序可交换。
Clairaut 定理的 前提是混合偏导连续。虽然大多数初等函数满足，但反例存在： \(f(x,y)=\frac{xy(x^2-y^2)}{x^2+y^2}\)（补定义 \(f(0,0)=0\)）有 \(f_{xy}(0,0)=-1\neq f_{yx}(0,0)=1\)。 考试只要求知道有这个前提。

  2. 验证 Clairaut 定理对 \(f(x,y)=x^4+y^4-4x^2y^2\) 成立。

14.4  切平面与全微分

▼切平面页2

曲面 \(z=f(x,y)\) 在点 \(P(x_0,y_0,z_0)\)（其中 \(z_0=f(x_0,y_0)\)）处的切平面：

\(\boxed{z-z_0 = f_x(x_0,y_0)(x-x_0) + f_y(x_0,y_0)(y-y_0)}\)

几何意义：用 切平面逼近曲面——曲面在切点附近"几乎平坦"。

▼线性逼近页2

线性化函数 \(L(x,y)\) 就是切平面的方程：
\(L(x,y)=f(x_0,y_0)+f_x(x_0,y_0)(x-x_0)+f_y(x_0,y_0)(y-y_0)\) 当 \((x,y)\) 接近 \((x_0,y_0)\) 时，\(f(x,y)\approx L(x,y)\)。

▼全微分页2

\(\boxed{dz = f_x\,dx + f_y\,dy = \frac{\partial z}{\partial x}dx + \frac{\partial z}{\partial y}dy}\)

全微分表示 线性主部：当 \(dx=\Delta x,\;dy=\Delta y\) 很小时，\(\Delta z\approx dz\)。
可微 ⇒ 连续 ⇒ 偏导存在（但偏导存在 ⇏ 可微，除非偏导连续）。

偏导数存在 ≠ 可微！经典反例： \(f(x,y)=\begin{cases}\frac{xy}{x^2+y^2},&(x,y)\neq(0,0)\\0,&(x,y)=(0,0)\end{cases}\) 在 \((0,0)\) 处偏导存在但不连续，从而不可微。

  2. 用全微分近似计算 \(\sqrt{4.02^2+3.01^2}\)。

14.5  链式法则

▼Case 1：中间变量都是一元函数

\(z=f(x,y)\)，其中 \(x=g(t),\,y=h(t)\)：
\(\boxed{\dfrac{dz}{dt}=\dfrac{\partial z}{\partial x}\dfrac{dx}{dt}+\dfrac{\partial z}{\partial y}\dfrac{dy}{dt}}\)

▼Case 2：中间变量都是二元函数

\(z=f(x,y)\)，其中 \(x=g(s,t),\,y=h(s,t)\)：
\(\dfrac{\partial z}{\partial s}=\dfrac{\partial z}{\partial x}\dfrac{\partial x}{\partial s}+\dfrac{\partial z}{\partial y}\dfrac{\partial y}{\partial s}\)
\(\dfrac{\partial z}{\partial t}=\dfrac{\partial z}{\partial x}\dfrac{\partial x}{\partial t}+\dfrac{\partial z}{\partial y}\dfrac{\partial y}{\partial t}\)

▼隐函数求导页5

若方程 \(F(x,y,z)=0\) 隐式定义 \(z=z(x,y)\)，则在 \(F_z\neq0\) 处：

\(\boxed{\dfrac{\partial z}{\partial x}=-\dfrac{F_x}{F_z},\qquad \dfrac{\partial z}{\partial y}=-\dfrac{F_y}{F_z}}\)

（由 \(F(x,y,z(x,y))=0\) 两边对 \(x\) 求偏导得 \(F_x+F_z\cdot z_x=0\) → 解出 \(z_x\)）

写链式法则树形图来避免漏项！画法：
  • 因变量在上，自变量在下
  • 每条有向边对应一个偏导因子
  • 每条从因变量到自变量的路径 = 一项，路径上的因子相乘，所有路径相加
自测 · 14.5

  2. \(z=f(x,y)\)，其中 \(x=u^2-v^2,\;y=2uv\)。用链式法则表达 \(\partial z/\partial u\) 和 \(\partial z/\partial v\)。
  3. \(F(x,y,z)=x^2+y^2+z^2-1=0\) 隐式定义 \(z=z(x,y)\)。求 \(\partial z/\partial x\) 和 \(\partial z/\partial y\)。

14.6  方向导数与梯度

▼方向导数页3

沿 单位向量 \(\mathbf{u}=(a,b)\)（\(a^2+b^2=1\)）的方向导数：

\(\boxed{D_{\mathbf{u}}f(x,y)=f_x(x,y)\,a+f_y(x,y)\,b = \nabla f\cdot\mathbf{u}}\)

▼梯度页3

\(\boxed{\nabla f(x,y) = \langle f_x(x,y),\;f_y(x,y)\rangle}\)

梯度的方向 = 函数增长最快的方向。
最大方向导数 = \(\boxed{|\nabla f|=\sqrt{f_x^2+f_y^2}}\)（沿梯度方向取到）。
最小方向导数 = \(-|\nabla f|\)（沿负梯度方向）。

▼梯度在几何中的应用

等位面 \(F(x,y,z)=k\) 的法向量：\(\nabla F(x_0,y_0,z_0)\)
等位面在 \(P_0\) 处的切平面： \(F_x(P_0)(x-x_0)+F_y(P_0)(y-y_0)+F_z(P_0)(z-z_0)=0\) 法线方程：\(\frac{x-x_0}{F_x(P_0)}=\frac{y-y_0}{F_y(P_0)}=\frac{z-z_0}{F_z(P_0)}\)

注：对曲面 \(z=f(x,y)\)（改写为 \(F(x,y,z)=f(x,y)-z=0\)），\(\nabla F=\langle f_x,f_y,-1\rangle\)， 此法向量代入切平面方程正好就是 14.4 节的公式。

方向导数定义中的 \(\mathbf{u}\) 必须是单位向量！如果题目给的是非单位向量，先归一化： \(\mathbf{u}=\frac{\mathbf{v}}{|\mathbf{v}|}\)。

  2. 求上题中 \((2,0)\) 处的梯度，以及最大方向导数。
  3. 求曲面 \(x^2+y^2+z^2=9\) 在 \((1,2,2)\) 处的切平面方程。

14.7  极值

▼临界点页2

若 \(f\) 在 \((a,b)\) 有局部极值且偏导存在，则 \(f_x(a,b)=0\) 且 \(f_y(a,b)=0\)。 满足此条件的点称为临界点（或驻点）。

临界点只是极值的候选，不一定真的是极值点。必须用二阶判别法进一步判断。

▼二阶判别法（Second Derivative Test）

设 \((a,b)\) 是 \(f\) 的临界点。定义： \(\boxed{D = f_{xx}(a,b)\,f_{yy}(a,b) - [f_{xy}(a,b)]^2}\)
\(D\)附加条件结论
\(D>0\)\(f_{xx}>0\)局部极小点
\(D>0\)\(f_{xx}<0\)局部极大点
\(D<0\)—鞍点
\(D=0\)—判别法失效
\(D>0\) 时 \(f_{xx}\) 和 \(f_{yy}\) 必同号，所以你只需要检查 \(f_{xx}\) 的符号即可。 若 \(f_{xx}=0\) 而 \(D>0\) 不可能发生（否则 \(D\le0\)）。

▼闭区域上的绝对最值

在有界闭区域 \(D\) 上求最值（连续函数必有最值）：

易错：当 D=0 判别法失效时，需要沿不同曲线（如 y=x, y=-x）代入 f，观察在临界点附近函数值的变化来判断极值还是鞍点。这是考试中 D=0 拔高题的标准处理方法。
  1. 找 内部临界点（\(f_x=f_y=0\)）并计算函数值
  2. 在 边界上化为单变量最值问题
  3. 比较所有候选点的函数值，取最大/最小
