## 核心公式速查

| 类型 | 解法 |
|------|------|
| 可分方程 dy/dx = g(x)h(y) | 分离变量 → ∫ dy/h(y) = ∫ g(x)dx |
| 一阶线性 y' + P(x)y = Q(x) | 积分因子 I(x) = e^(∫P dx) → (I·y)' = I·Q → y = (∫IQ dx + C)/I |
