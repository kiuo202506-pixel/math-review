## 14.1 多元函数基本概念

**1.** 定义域：$\{(x,y): x^2+y^2 \le 9\}$（半径为 3 的圆盘）。值域：$[0,3]$。图像：上半球面。

**2.** $x^2+y^2-4 > 0$ → $\{(x,y): x^2+y^2 > 4\}$（半径 2 的圆外部）。

**3.** $k=1$：圆 $x^2+y^2=1$；$k=4$：圆 $x^2+y^2=4$；$k=9$：圆 $x^2+y^2=9$。这些是以原点为中心的同心圆。

**4.** 等高线 $xy=k$ 为双曲线族（$k\neq0$），$k=0$ 时为坐标轴。

## 14.2 极限与连续性

**5.** 代入：$f(1,2) = 1^2\cdot2 + 3\cdot1\cdot4 - 2 = 2+12-2 = 12$

**6.** 沿 $y=0$：$\lim\frac{0}{x^2}=0$。沿 $y=x$：$\lim\frac{x^2}{2x^2}=\frac{1}{2}$。两路径不同 → 极限不存在。

**7.** 沿 $y=0$：$\lim\frac{x^2}{x^2}=1$。沿 $x=0$：$\lim\frac{-y^2}{y^2}=-1$。极限不存在。

**8.** 用极坐标：$x=r\cos\theta$，$y=r\sin\theta$，$f=\frac{r^3\cos^2\theta\sin\theta}{r^2}=r\cos^2\theta\sin\theta$。$r\to0$ 时 $f\to0$（有界乘零因子）。极限 $=0$。

**9.** $f$ 在 $(0,0)$ 处不连续——极限不存在（题 7），因此 $f(0,0)=0$ 不等于极限值。

## 14.3 偏导数

**10.** $f_x = 3x^2y + 6xy^2 - 4y^3$，$f_y = x^3 + 6x^2y - 12xy^2$

**11.** $f_x = ye^{xy}\sin(x+y) + e^{xy}\cos(x+y)$，$f_y = xe^{xy}\sin(x+y) + e^{xy}\cos(x+y)$

**12.** $f_x = \ln(yz) + \frac{y\sqrt{z}}{2\sqrt{x}}$，$f_y = \frac{x}{y} + \sqrt{xz}$，$f_z = \frac{x}{z} + \frac{y\sqrt{x}}{2\sqrt{z}}$

**13.** $f_x = 4x^3y^2 - 4xy^3$，$f_{xx} = 12x^2y^2 - 4y^3$。$f_y = 2x^4y - 6x^2y^2 + 8$，$f_{yy} = 2x^4 - 12x^2y$。$f_{xy} = 8x^3y - 12xy^2$，$f_{yx} = 8x^3y - 12xy^2$，相等。

**14.** $f_x = \frac{2x}{x^2+y^2}$，$f_y = \frac{2y}{x^2+y^2}$。$f_{xy} = -\frac{4xy}{(x^2+y^2)^2}$，$f_{yx} = -\frac{4xy}{(x^2+y^2)^2}$，相等。

## 14.4 切平面与全微分

**15.** $f_x=2x$，$f_y=2y$，在 $(1,1)$：$f_x=2$，$f_y=2$。切平面：$z-2=2(x-1)+2(y-1)$ → $z=2x+2y-2$

**16.** $f(x,y)=\ln(x-2y)$，$f_x=\frac{1}{x-2y}$，$f_y=\frac{-2}{x-2y}$。$f(3,1)=\ln 1=0$，$f_x(3,1)=1$，$f_y(3,1)=-2$。$L(x,y)=0+1(x-3)-2(y-1)=x-2y-1$。$f(3.02,0.97)\approx L(3.02,0.97)=3.02-2(0.97)-1=0.08$

**17.** $dz = (3x^2y^2+4y)dx + (2x^3y+4x)dy$

**18.** $dV = 2\pi rh\,dr + \pi r^2\,dh$。$|\Delta V| \approx |2\pi\cdot10\cdot20\cdot0.1 + \pi\cdot100\cdot0.1| = |40\pi + 10\pi| = 50\pi \approx 157.1$ cm³

## 14.5 链式法则

**19.** $\frac{dz}{dt} = \frac{\partial z}{\partial x}\frac{dx}{dt} + \frac{\partial z}{\partial y}\frac{dy}{dt} = (2xy+3y^4)(2\cos 2t) + (x^2+12xy^3)(-\sin t)$。代入 $x=\sin 2t$，$y=\cos t$。

**20.** $\frac{\partial z}{\partial s} = e^x\sin y\cdot t^2 + e^x\cos y\cdot 2st$。$\frac{\partial z}{\partial t} = e^x\sin y\cdot 2st + e^x\cos y\cdot s^2$

**21.** 设 $F(x,y,z) = y^3z + xz^2 - e^z = 0$。$F_x = z^2$，$F_y = 3y^2z$，$F_z = y^3 + 2xz - e^z$。$\frac{\partial z}{\partial x} = -\frac{F_x}{F_z} = \frac{-z^2}{y^3+2xz-e^z}$，$\frac{\partial z}{\partial y} = -\frac{F_y}{F_z} = \frac{-3y^2z}{y^3+2xz-e^z}$

## 14.6 方向导数与梯度

**22.** $\nabla f = \langle 2xy^3, 3x^2y^2-4y^3 \rangle$。在 $(2,1)$：$\nabla f = \langle 4, 12-4 \rangle = \langle 4, 8 \rangle$。$\vec{u} = \langle 3/5, 4/5 \rangle$。$D_uf = 4\cdot\frac{3}{5} + 8\cdot\frac{4}{5} = \frac{44}{5}$

**23.** $f(x,y,z)=x\ln(yz)$，$\nabla f = \langle \ln(yz), x/y, x/z \rangle$。在 $(1,2,1)$：$\nabla f = \langle \ln 2, 1/2, 1 \rangle$。$\vec{u} = \langle 2/3, 1/3, -2/3 \rangle$。$D_uf = \ln 2\cdot\frac{2}{3} + \frac{1}{2}\cdot\frac{1}{3} + 1\cdot(-\frac{2}{3}) = \frac{2}{3}\ln 2 - \frac{1}{2}$

**24.** $\nabla f = \langle 2x, 2y \rangle$，在 $(3,4)$：$\langle 6, 8 \rangle$

**25.** $\nabla f = \langle \cos x + ye^{xy}, xe^{xy} \rangle$，在 $(0,1)$：$\langle 1+1, 0 \rangle = \langle 2, 0 \rangle$

**26.** 最大方向导数方向为梯度方向 $\langle 6,8 \rangle$，单位化 $\langle 3/5, 4/5 \rangle$。最大值 $= |\nabla f| = 10$

**27.** $\nabla f = \langle yz, xz, xy \rangle$，在 $(1,2,3)$：$\langle 6, 3, 2 \rangle$。最大方向导数方向为梯度方向，最大值 $=\sqrt{36+9+4}=7$

**28.** $F(x,y,z)=x^2+y^2+z^2-9$，$\nabla F = \langle 2x,2y,2z \rangle$，在 $(2,2,1)$：$\langle 4,4,2 \rangle$。切平面：$4(x-2)+4(y-2)+2(z-1)=0$ → $2x+2y+z=9$

**29.** $z=f(x,y)=x^2-y^2$，切平面 $z = f(1,1)+f_x(1-x)+f_y(1-y) = 0+2(x-1)-2(y-1) = 2x-2y$。法线方向 $\langle f_x, f_y, -1 \rangle = \langle 2, -2, -1 \rangle$。法线方程：$\frac{x-1}{2} = \frac{y-1}{-2} = \frac{z}{-1}$

## 14.7 极值与二阶判别法

**30.** $f_x=2x+y+3=0$，$f_y=x+2y-3=0$。解得 $x=-3$，$y=3$。$f_{xx}=2$，$f_{yy}=2$，$f_{xy}=1$，$D=4-1=3>0$，$f_{xx}>0$ → 局部极小 $f(-3,3)=-13$

**31.** $f_x=y-2-2x=0$，$f_y=x-2-2y=0$。解得 $x=-2$，$y=-2$。$f_{xx}=f_{yy}=-2$，$f_{xy}=1$，$D=4-1=3>0$，$f_{xx}<0$ → 局部极大 $f(-2,-2)=4$

**32.** $f_x=6x^2+y^2-12x=0$，$f_y=2xy+2y=2y(x+1)=0$。由 $f_y=0$：$y=0$ 或 $x=-1$。$y=0$ 时：$6x^2-12x=0$ → $x=0,2$。$x=-1$ 时：$6+y^2+12=0$ 无解。临界点：$(0,0)$，$(2,0)$。$f_{xx}=12x-12$，$f_{yy}=2x+2$，$f_{xy}=2y$。$(0,0)$：$D=-24<0$ → 鞍点。$(2,0)$：$D=12\cdot6-0=72>0$，$f_{xx}=12>0$ → 局部极小。

**33.** $f_x=3e^y-3x^2=0$ → $e^y=x^2$。$f_y=3xe^y-3e^{3y}=0$ → $x=e^{2y}$。代入：$e^y=e^{4y}$ → $y=0$，$x=1$。唯一临界点 $(1,0)$。$D(1,0)$ 计算得 $<0$ → 鞍点。但检查：当 $x\to\infty$ 时 $f\to-\infty$（$x^3$ 主导），当 $y\to\infty$ 时 $f\to-\infty$（$e^{3y}$ 主导）。$(1,0)$ 是鞍点，函数无局部极值，无全局最大值。

**34.** 内部临界点：$f_x=2x-2=0$，$f_y=2y=0$ → $(1,0)$。$f(1,0)=-1$。边界 $x^2+y^2=4$：参数化 $x=2\cos t$，$y=2\sin t$，$f=4-4\cos t$，$t=0$ 时 $f=0$（最小），$t=\pi$ 时 $f=8$。绝对最大值 $8$（$(-2,0)$），绝对最小值 $-1$（$(1,0)$）。

**35.** $f_x=6x^2=0$ → $x=0$。$f_y=4y^3=0$ → $y=0$。内部仅有 $(0,0)$，$f(0,0)=0$。边界 $x^2+y^2=1$：$f=2x^3+(1-x^2)^2 = 2x^3+1-2x^2+x^4$。$f'(x)=6x^2-4x+4x^3$，解数值 $x\approx -0.65$ 时 $f\approx -0.46$。$x=1$ 时 $f=2$，$x=-1$ 时 $f=-1$。绝对最大值 $=2$（$(1,0)$），绝对最小值 $\approx -1$（或 $-1$ 在 $(-1,0)$）。
