## 8.1 弧长

**1.** $y=2x-5$，$y'=2$，$\sqrt{1+4}=\sqrt{5}$，$L = \int_0^3 \sqrt{5}\,dx = 3\sqrt{5}$

**2.** $y = 1+6x^{3/2}$，$y' = 9x^{1/2}$，$L = \int_0^1 \sqrt{1+81x}\,dx$。令 $u=1+81x$，$du=81dx$：$L = \frac{1}{81}\int_1^{82} \sqrt{u}\,du = \frac{2}{243}(82^{3/2}-1)$

**3.** $y = \frac{2}{3}(x^2+1)^{3/2}$，$y' = 2x\sqrt{x^2+1}$，$\sqrt{1+(y')^2} = \sqrt{1+4x^2(x^2+1)} = 2x^2+1$，$L = \int_0^1 (2x^2+1)dx = \frac{5}{3}$

**4.** $y^2 = 4(x+4)^3$，$2y\,y' = 12(x+4)^2$，$y' = \frac{6(x+4)^2}{y}$。用 $y$ 做自变量：$x = \frac{1}{4}y^{2/3} - 4$，$dx/dy = \frac{1}{6}y^{-1/3}$。$\sqrt{1+(dx/dy)^2}$ 形式更简。在 $x=0$ 时 $y=16$，$x=2$ 时 $y=24\sqrt{6}$。$L \approx$ 数值积分。

**5.** $y = \ln(\sec x)$，$y' = \tan x$，$\sqrt{1+\tan^2 x} = |\sec x| = \sec x$（在第一象限）。$L = \int_0^{\pi/4} \sec x\,dx = \ln|\sec x+\tan x|\big|_0^{\pi/4} = \ln(\sqrt{2}+1)$

**6.** $y = \frac{1}{4}x^2 - \frac{1}{2}\ln x$，$y' = \frac{x}{2} - \frac{1}{2x}$，$(y')^2 = \frac{x^2}{4} - \frac{1}{2} + \frac{1}{4x^2}$，$1+(y')^2 = \frac{x^2}{4} + \frac{1}{2} + \frac{1}{4x^2} = \left(\frac{x}{2} + \frac{1}{2x}\right)^2$，$L = \int_1^2 \left(\frac{x}{2} + \frac{1}{2x}\right)dx = \frac{3}{4} + \frac{1}{2}\ln 2$

**7.** $x=t^2$，$y=2t$，$dx/dt=2t$，$dy/dt=2$，$L = \int_0^{\sqrt{3}} \sqrt{4t^2+4}\,dt = 2\int_0^{\sqrt{3}} \sqrt{t^2+1}\,dt = t\sqrt{t^2+1}+\ln|t+\sqrt{t^2+1}|\big|_0^{\sqrt{3}} = 2\sqrt{3}+\ln(2+\sqrt{3})$

**8.** $x=e^t\cos t$，$y=e^t\sin t$，$dx/dt = e^t(\cos t-\sin t)$，$dy/dt = e^t(\sin t+\cos t)$，$(dx/dt)^2+(dy/dt)^2 = e^{2t}[(\cos t-\sin t)^2+(\sin t+\cos t)^2] = 2e^{2t}$，$L = \int_0^\pi \sqrt{2}e^t\,dt = \sqrt{2}(e^\pi-1)$

**9.** $f'(x) = x^{1/2}$，$\sqrt{1+x} = \sqrt{1+x}$。$s(x) = \int_1^x \sqrt{1+t}\,dt = \frac{2}{3}[(1+x)^{3/2} - 2^{3/2}]$

**10.** $f'(x) = \sqrt{3x+5}$，$f(x) = \int \sqrt{3x+5}\,dx = \frac{2}{9}(3x+5)^{3/2} + C$。由 $f(0)=2$：$\frac{2}{9}\cdot 5^{3/2}+C=2$，$C=2-\frac{10\sqrt{5}}{9}$。$f(x) = \frac{2}{9}(3x+5)^{3/2} + 2 - \frac{10\sqrt{5}}{9}$

## 8.2 旋转曲面面积

**11.** 绕 $x$ 轴：$S_x = \int_1^2 2\pi x^{-2}\sqrt{1+(-2x^{-3})^2}\,dx = 2\pi\int_1^2 x^{-2}\sqrt{1+4x^{-6}}\,dx$。数值积分 ≈ $4.4566$

**12.** 绕 $y$ 轴：$S_y = \int_1^2 2\pi x\sqrt{1+4x^{-6}}\,dx$。或写为 $x = y^{-1/2}$，$1\le y\le 1/4$。数值积分 ≈ $11.7299$

**13.** $S = \int_0^\pi 2\pi\sin x\sqrt{1+\cos^2 x}\,dx$。令 $u=\cos x$，$du=-\sin x\,dx$：$S = 2\pi\int_{-1}^1 \sqrt{1+u^2}\,du = 4\pi\int_0^1 \sqrt{1+u^2}\,du$。$\int \sqrt{1+u^2}\,du = \frac{u}{2}\sqrt{1+u^2}+\frac{1}{2}\ln|u+\sqrt{1+u^2}|$。$S = 2\pi[\sqrt{2}+\ln(1+\sqrt{2})]$

**14.** $y = \cos(x/2)$，$y' = -\frac{1}{2}\sin(x/2)$，$S = \int_0^\pi 2\pi\cos(x/2)\sqrt{1+\frac{1}{4}\sin^2(x/2)}\,dx = 4\pi\int_0^{\pi/2} \cos t\sqrt{1+\frac{1}{4}\sin^2 t}\,dt$（$t=x/2$）。再令 $u=\sin t$：$S = 4\pi\int_0^1 \sqrt{1+\frac{u^2}{4}}\,du$。数值积分 ≈ …

**15.** $y=x^3$，$y'=3x^2$，$S = \int_0^1 2\pi x^3\sqrt{1+9x^4}\,dx$。令 $u=1+9x^4$，$du=36x^3\,dx$：$S = \frac{\pi}{18}\int_1^{10} \sqrt{u}\,du = \frac{\pi}{27}(10^{3/2}-1)$

**16.** 球面由 $y=\sqrt{r^2-x^2}$（上半圆）绕 $x$ 轴旋转生成。$y' = -x/\sqrt{r^2-x^2}$，$\sqrt{1+(y')^2} = r/\sqrt{r^2-x^2}$。$S = \int_{-r}^r 2\pi\sqrt{r^2-x^2}\cdot\frac{r}{\sqrt{r^2-x^2}}\,dx = 4\pi r^2$

## 8.3 物理与工程应用

**17.** 建立坐标系：水面在 $y=0$，坝底在 $y=30$。深度 $d(y)=y$（水面在坝顶上方 4m 处）。梯形宽度：$w(y)=20+\frac{20}{30}y$（从底宽 20m 线性变化到顶宽 40m）。$F = \int_4^{30} \rho g\cdot y\cdot w(y)\,dy = \rho g\int_4^{30} y(20+\frac{2}{3}y)dy$。代入 $\rho g = 9800$ N/m³ 得数值。

**18.** $M = 6\cdot10 + 9\cdot30 = 330$，$m = 6+9 = 15$，$\bar{x} = 330/15 = 22$

**19.** $A = \int_0^4 \sqrt{x}\,dx = \frac{16}{3}$。$\bar{x} = \frac{1}{A}\int_0^4 x\sqrt{x}\,dx = \frac{1}{A}\int_0^4 x^{3/2}dx = \frac{1}{A}\cdot\frac{128}{5} = \frac{12}{5}$。$\bar{y} = \frac{1}{A}\int_0^4 \frac{1}{2}x\,dx = \frac{1}{A}\cdot4 = \frac{3}{4}$。形心 $(\frac{12}{5}, \frac{3}{4})$

**20.** $\bar{x} = \pi/2$（由对称性）。$A = \int_0^\pi \sin x\,dx = 2$。$\bar{y} = \frac{1}{A}\int_0^\pi \frac{1}{2}\sin^2 x\,dx = \frac{1}{4}\int_0^\pi (1-\cos 2x)dx = \frac{\pi}{8}$。形心 $(\frac{\pi}{2}, \frac{\pi}{8})$

**21.** 圆环体：将半径为 $r$ 的圆绕距离圆心 $R$ 的外部直线旋转。由 Pappus 定理，体积 $V = 2\pi\cdot R\cdot \pi r^2 = 2\pi^2 R r^2$（绕 $y$ 轴的圈面旋转所得。）
