# Ch10 · 参数方程与极坐标 — 参考答案

---

## 10.1 参数方程

**1.** 由 $x=3t-2$ 得 $t=(x+2)/3$，代入 $y=2t+5$：$y=\frac{2}{3}(x+2)+5=\frac{2}{3}x+\frac{19}{3}$

**2.** $x=\cos t$，$y=\sin^2 t = 1-\cos^2 t = 1-x^2$，$-1\le x\le 1$

**3.** $x=\frac{1}{2}\cos t$，$y=\frac{1}{5}\sin t$（$0\le t\le 2\pi$）

---

## 10.2 参数曲线微积分

**4.** $dx/dt=3t^2$，$dy/dt=2t-2$。$t=1$ 时：$(x,y)=(1,-1)$。$dy/dx=(2t-2)/(3t^2)$。$t=1$ 时斜率为 $0$。切线：$y=-1$

**5.** 由 $(0,-1)$ 得 $t=1$。$dy/dx=(2t-2)/(1/t)=t(2t-2)$。$t=1$ 时斜率 $=0$。切线：$y=-1$

**6.** $L=\int_0^4 \sqrt{3^2+(3t^{1/2})^2}\,dt = \int_0^4 \sqrt{9+9t}\,dt = 3\int_0^4 \sqrt{1+t}\,dt = 2(1+t)^{3/2}\big|_0^4 = 2(5^{3/2}-1)$

**7.** $dx/dt=e^t-e^{-t}$，$dy/dt=-2$，$L=\int_0^3 \sqrt{(e^t-e^{-t})^2+4}\,dt = \int_0^3 (e^t+e^{-t})dt = (e^t-e^{-t})\big|_0^3 = (e^3-e^{-3})$

**8.** $x=t^2$，$y=t^3$，$t\ge 0$。$x=1$ 时 $t=1$。$A=\int_0^1 y(t)x'(t)dt = \int_0^1 t^3\cdot 2t\,dt = 2\int_0^1 t^4dt = \frac{2}{5}$

**9.** $A=\int_0^{2\pi} (1-\cos t)(1-\cos t)dt = \int_0^{2\pi} (1-\cos t)^2dt = \int_0^{2\pi} (1-2\cos t+\cos^2 t)dt = \int_0^{2\pi} (1-2\cos t+\frac{1+\cos 2t}{2})dt = \int_0^{2\pi} (\frac{3}{2}-2\cos t+\frac{\cos 2t}{2})dt = 3\pi$

---

## 10.3 极坐标

**10.** $x=2\cos(\pi/3)=1$，$y=2\sin(\pi/3)=\sqrt{3}$，$(1,\sqrt{3})$

**11.** $r=\sqrt{(-3)^2+3^2}=3\sqrt{2}$，$\tan\theta=-1$ 在第二象限，$\theta=3\pi/4$。$(3\sqrt{2}, 3\pi/4)$

**12.** 心脏线 $r=1+\cos\theta$：$\theta=0$ 时 $r=2$；$\theta=\pi/2$ 时 $r=1$；$\theta=\pi$ 时 $r=0$，关于极轴呈心形对称。\
**13.** 玫瑰线 $r=\sin 2\theta$：当 $\sin 2\theta\ge0$ 时 $r$ 有定义，在 $0\le\theta\le\pi/2$、$\pi\le\theta\le3\pi/2$ 各得一瓣；$\sin 2\theta<0$ 的两段与之重合（曲线总共四瓣）。

**14.** $r(\theta)=\cos(\theta/3)$，$r'(\theta)=-\frac{1}{3}\sin(\theta/3)$。$\theta=\pi$ 时 $r=\cos(\pi/3)=1/2$，$r'=-\frac{1}{3}\sin(\pi/3)=-\frac{\sqrt{3}}{6}$。$\frac{dy}{dx}=\frac{r'\sin\theta+r\cos\theta}{r'\cos\theta-r\sin\theta}$ 代入得 $\frac{dy}{dx}=-\sqrt{3}$。$x=r\cos\theta=-1/2$，$y=r\sin\theta=0$。切线：$y=-\sqrt{3}(x+1/2)$

---

## 10.4 极坐标面积与弧长

**15.** $r=\sin 4\theta$，一瓣对应 $0\le\theta\le\pi/4$。$A=\int_0^{\pi/4} \frac{1}{2}\sin^2 4\theta\,d\theta = \frac{1}{2}\int_0^{\pi/4} \frac{1-\cos 8\theta}{2}d\theta = \frac{\pi}{16}$

**16.** $A=\int_0^{2\pi} \frac{1}{2}(1+\cos\theta)^2 d\theta = \frac{1}{2}\int_0^{2\pi} (1+2\cos\theta+\cos^2\theta)d\theta = \frac{1}{2}\int_0^{2\pi} (1+2\cos\theta+\frac{1+\cos 2\theta}{2})d\theta = \frac{1}{2}\int_0^{2\pi} \frac{3}{2}d\theta = \frac{3\pi}{2}$

**17.** $r^2=4\cos 2\theta$，一环比 $\cos 2\theta\ge 0$ 即 $-\pi/4\le\theta\le\pi/4$。$A=\frac{1}{2}\int_{-\pi/4}^{\pi/4} 4\cos 2\theta\,d\theta = \int_{-\pi/4}^{\pi/4} 2\cos 2\theta\,d\theta = \sin 2\theta\big|_{-\pi/4}^{\pi/4} = 2$

**18.** 两圆交点：$(x-1)^2+y^2=1$ → $r=2\cos\theta$；$x^2+y^2=1$ → $r=1$。交点满足 $2\cos\theta=1$ → $\theta=\pm\pi/3$。$A=2\int_0^{\pi/3} \frac{1}{2}[(2\cos\theta)^2-1^2]d\theta = \int_0^{\pi/3} (4\cos^2\theta-1)d\theta = \int_0^{\pi/3} (2+2\cos 2\theta-1)d\theta = [\theta+\sin 2\theta]_0^{\pi/3} = \frac{\pi}{3}+\frac{\sqrt{3}}{2}$

**19.** $r'=5^\theta\ln 5$，$r^2+(r')^2=5^{2\theta}(1+\ln^2 5)$。$L=\int_0^\pi 5^\theta\sqrt{1+\ln^2 5}\,d\theta = \sqrt{1+\ln^2 5}\cdot\frac{5^\theta}{\ln 5}\big|_0^\pi = \frac{\sqrt{1+\ln^2 5}}{\ln 5}(5^\pi-1)$

**20.** $r=1+\cos\theta$，$dr/d\theta=-\sin\theta$。$r^2+(dr/d\theta)^2=(1+\cos\theta)^2+\sin^2\theta = 2+2\cos\theta = 4\cos^2(\theta/2)$。$L=\int_0^{2\pi} 2|\cos(\theta/2)|d\theta = 4\int_0^\pi \cos(\theta/2)d\theta = 8$
