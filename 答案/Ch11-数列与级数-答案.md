# Ch11 · 数列与级数 — 参考答案

---

## 11.1 数列

**1.** $\displaystyle\lim_{n\to\infty}\frac{2n}{3n+1} = \frac{2}{3}$

**2.** $\displaystyle\lim_{n\to\infty}\frac{n^2-1}{n^2+n} = 1$

**3.** $\displaystyle\lim_{n\to\infty}\sqrt{\frac{n+4}{9n+1}} = \sqrt{\frac{1}{9}} = \frac{1}{3}$

**4.** $\displaystyle\lim_{n\to\infty}\left(1+\frac{2}{n}\right)^n = e^2$

**5.** $|a_n| = \frac{n}{n+1} \to 1 \neq 0$，发散。

**6.** $a_n > 0$，$a_{n+1}/a_n = \frac{n+1}{(n+1)^2+1} \cdot \frac{n^2+1}{n} < 1$（递减）；$a_n \to 0$，有界（最小值 $0$，最大值 $a_1=1/2$）。

**7.** $a_{n+1} = \frac{1}{2}(a_n+2/a_n) \ge \sqrt{a_n\cdot 2/a_n} = \sqrt{2}$（AM-GM）。数列递减（对 $a_n>\sqrt{2}$而言）且下有界，收敛。设极限为 $L$，则 $L=\frac{1}{2}(L+2/L)$ → $L=\sqrt{2}$。

## 11.2 级数

**8.** $\sum_{n=1}^{\infty} \frac{2}{3^{n-1}} = 2\sum_{n=0}^{\infty} \left(\frac{1}{3}\right)^n = \frac{2}{1-1/3} = 3$

**9.** $\sum_{n=0}^{\infty} \frac{(-5)^n}{4^{n+1}} = \frac{1}{4}\sum_{n=0}^{\infty} \left(-\frac{5}{4}\right)^n$，$|r|=5/4 > 1$，发散。

**10.** $\frac{1}{n(n+1)} = \frac{1}{n} - \frac{1}{n+1}$。$s_N = 1 - \frac{1}{N+1} \to 1$

**11.** $\frac{1}{n^2-1} = \frac{1}{2}\left(\frac{1}{n-1} - \frac{1}{n+1}\right)$。$s_N = \frac{1}{2}\left(1+\frac{1}{2} - \frac{1}{N} - \frac{1}{N+1}\right) \to \frac{3}{4}$

**12.** $\displaystyle\lim_{n\to\infty}\frac{n^2}{2n^2+1} = \frac{1}{2} \neq 0$，发散。

**13.** $\displaystyle\lim_{n\to\infty}\frac{2n}{3n+1} = \frac{2}{3} \neq 0$，发散。

**14.** $\int_1^\infty \frac{1}{x}dx$ 发散 → 调和级数发散（积分审敛法）。

---

## 11.3 积分审敛法

**15.** $f(x)=\frac{1}{x^2+1}$ 正、连续、递减。$\int_1^\infty \frac{dx}{x^2+1} = \arctan x\big|_1^\infty = \frac{\pi}{2} - \frac{\pi}{4} = \frac{\pi}{4}$，收敛。

**16.** $\int_2^\infty \frac{dx}{x\ln x} = \ln|\ln x|\big|_2^\infty$，发散。

**17.** $\int_1^\infty \frac{x}{x^2+1}dx = \frac{1}{2}\ln(x^2+1)\big|_1^\infty$，发散。

**18.** $\sum 1/n^3$：$p=3>1$，收敛。$\sum 1/\sqrt{n}$：$p=1/2\le 1$，发散。

**19.** $R_{10} \le \int_{10}^\infty \frac{dx}{x^4} = \frac{1}{3\cdot 10^3} = \frac{1}{3000} \approx 0.00033$

---

## 11.4 比较审敛法

**20.** $\frac{1}{n^2+3} < \frac{1}{n^2}$，$\sum 1/n^2$ 收敛（p-级数 $p=2$），由比较审敛法收敛。

**21.** $\frac{1}{\sqrt{n^3+1}} < \frac{1}{n^{3/2}}$，$\sum 1/n^{3/2}$ 收敛（$p=1.5>1$），收敛。

**22.** $a_n = \frac{1}{2^n-1}$，$b_n = \frac{1}{2^n}$。$\lim a_n/b_n = \lim \frac{2^n}{2^n-1} = 1$。$\sum 1/2^n$ 收敛（几何级数 $r=1/2$），由极限比较审敛法收敛。

**23.** $a_n = \frac{n^2-1}{n^3+2n^2+5} \sim \frac{1}{n}$。取 $b_n = 1/n$，$\lim a_n/b_n = 1$。$\sum 1/n$ 发散，极限比较审敛法 → 发散。

**24.** 当 $n$ 大时，$\sin(1/n) \sim 1/n$，故 $a_n \sim \frac{1}{n^3}$。与 $\sum 1/n^3$（$p=3$，收敛）极限比较，收敛。

## 11.5 交错级数

**25.** $b_n = 1/n$，递减且 $\to 0$。交错级数审敛法 → 收敛（条件收敛）。

**26.** $b_n = \frac{n}{n^2+1} = \frac{1}{n+1/n}$，递减且 $\to 0$。收敛。

**27.** $f(x)=\ln x/x$，$f'(x)=(1-\ln x)/x^2 < 0$ 当 $x>e$。$n\ge 3$ 后递减，$\lim \ln n/n = 0$。收敛。

**28.** $|R_n| \le b_{n+1} = \frac{1}{(n+1)^5} < 0.0001$ → $(n+1)^5 > 10^4$ → $n+1 > 10^{4/5} \approx 6.31$ → $n \ge 6$。需要 6 项。

**29.** $|R_6| \le b_7 = \frac{1}{7!} = \frac{1}{5040} \approx 0.000198$
