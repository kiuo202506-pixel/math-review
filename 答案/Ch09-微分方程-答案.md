## 9.3 可分方程

**1.** $\frac{dy}{dx}=3x^2y^2$ → $\frac{dy}{y^2}=3x^2dx$ → $-\frac{1}{y}=x^3+C$ → $y=-\frac{1}{x^3+C}$

**2.** $\frac{dy}{dx}=x\sqrt{y}$ → $\frac{dy}{\sqrt{y}}=x\,dx$ → $2\sqrt{y}=\frac{x^2}{2}+C$ → $y=\left(\frac{x^2}{4}+C\right)^2$

**3.** $xy\frac{dy}{dx}=x^2+1$ → $y\,dy=\frac{x^2+1}{x}dx$ → $\frac{y^2}{2}=x+\frac{1}{x}+C$ → $y=\pm\sqrt{2x+\frac{2}{x}+2C}$

**4.** $(e^y-1)\frac{dy}{dx}=2+\cos x$ → $(e^y-1)dy=(2+\cos x)dx$ → $e^y-y=2x+\sin x+C$

**5.** $\frac{dp}{dt}=t^2p-p+t^2-1 = (p+1)(t^2-1)$ → $\frac{dp}{p+1}=(t^2-1)dt$ → $\ln|p+1|=\frac{t^3}{3}-t+C$ → $p=-1+Ke^{t^3/3-t}$

**6.** $e^{-y}\sin x - y'\cos^2 x = 0$ → $\frac{dy}{dx}=e^{-y}\frac{\sin x}{\cos^2 x}$ → $e^y\,dy=\frac{\sin x}{\cos^2 x}dx$ → $e^y=\sec x+C$。$y(0)=0$ → $1=1+C$ → $C=0$。$\therefore\; y=\ln|\sec x|$

**7.** $\frac{dy}{dx}=\frac{x^2}{y(1+x^3)}$ → $y\,dy=\frac{x^2}{1+x^3}dx$ → $\frac{y^2}{2}=\frac{1}{3}\ln|1+x^3|+C$。$y(0)=1$ → $\frac{1}{2}=C$。$y=\sqrt{\frac{2}{3}\ln(1+x^3)+1}$

**8.** 牛顿冷却：$\frac{dT}{dt}=-k(T-20)$。分离变量：$\ln|T-20|=-kt+C$ → $T=20+Ce^{-kt}$。$T(0)=95$ → $C=75$。$T(1)=90$ → $70=75e^{-k}$ → $e^{-k}=\frac{14}{15}$。$T(t)=20+75\left(\frac{14}{15}\right)^t$。$T(5)=20+75\left(\frac{14}{15}\right)^5 \approx 71.9$°C

**9.** $\frac{dy}{dt} = 3\cdot2 - 2\cdot\frac{y}{100} = 6 - \frac{y}{50}$。$\frac{dy}{dt}+\frac{y}{50}=6$。$I(t)=e^{t/50}$。$\frac{d}{dt}[ye^{t/50}]=6e^{t/50}$ → $ye^{t/50}=300e^{t/50}+C$ → $y=300+Ce^{-t/50}$。$y(0)=0$ → $C=-300$。$y(t)=300(1-e^{-t/50})$

## 9.5 一阶线性方程

**10.** $y'+2y=1$，$I=e^{2x}$，$ye^{2x}=\int e^{2x}dx=\frac{1}{2}e^{2x}+C$，$y=\frac{1}{2}+Ce^{-2x}$。$y(0)=2$ → $C=\frac{3}{2}$。$y=\frac{1}{2}+\frac{3}{2}e^{-2x}$

**11.** $y'-\frac{1}{2}y=1$，$I=e^{-x/2}$，$y=\frac{\int e^{-x/2}dx}{e^{-x/2}}=-2+Ce^{x/2}$。$y(0)=3$ → $C=5$。$y=5e^{x/2}-2$

**12.** $y'+\frac{2x}{1+x^2}y=15x^2$，$I=e^{\ln(1+x^2)}=1+x^2$。$y(1+x^2)=\int 15x^2(1+x^2)dx=5x^3+3x^5+C$。$y(1)=6$ → $6\cdot2=5+3+C$ → $C=4$。$y=\frac{5x^3+3x^5+4}{1+x^2}$

**13.** $y'+\frac{1}{x}y=1$，$I=x$，$xy=\int x\,dx=\frac{x^2}{2}+C$，$y=\frac{x}{2}+\frac{C}{x}$。$y(4)=5$ → $5=2+\frac{C}{4}$ → $C=12$。$y=\frac{x}{2}+\frac{12}{x}$

**14.** $y'+\frac{2}{x}y=7\sqrt{x}$，$I=x^2$，$x^2y=\int 7x^{5/2}dx=2x^{7/2}+C$，$y=2x^{3/2}+\frac{C}{x^2}$。$y(1)=9$ → $9=2+C$ → $C=7$。$y=2x^{3/2}+\frac{7}{x^2}$

**15.** $y'+xy=x$，$I=e^{x^2/2}$，$ye^{x^2/2}=\int xe^{x^2/2}dx=e^{x^2/2}+C$，$y=1+Ce^{-x^2/2}$。$y(0)=4$ → $C=3$。$y=1+3e^{-x^2/2}$

**16.** $y'-2xy=4x$，$I=e^{-x^2}$，$ye^{-x^2}=\int 4xe^{-x^2}dx=-2e^{-x^2}+C$，$y=-2+Ce^{x^2}$。$y(0)=1$ → $1=-2+C$ → $C=3$。$y=3e^{x^2}-2$

**17.** $y'-\cot x\cdot y=\csc x$，$P(x)=-\cot x$，$\int -\cot x\,dx = -\ln|\sin x|$，$I = e^{-\ln|\sin x|} = \csc x$。$y\csc x = \int \csc^2 x\,dx = -\cot x + C$。$y = -\cos x + C\sin x$。$y(\pi/2)=4$ → $4=0+C$ → $C=4$。$y=4\sin x-\cos x$
