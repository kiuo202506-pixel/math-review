# Ch17 · 二阶微分方程 — 参考答案

---

## A 组 · 通解

**1.** $r^2+2r-3=0$，$(r+3)(r-1)=0$，$r=-3,1$。$y=c_1e^{-3x}+c_2e^x$

**2.** $r^2-6r+9=0$，$(r-3)^2=0$，$r=3$（重根）。$y=c_1e^{3x}+c_2xe^{3x}$

**3.** $r^2+4r+13=0$，$r=-2\pm 3i$。$y=e^{-2x}(c_1\cos 3x + c_2\sin 3x)$

**4.** $4r^2+4r+1=0$，$(2r+1)^2=0$，$r=-1/2$（重根）。$y=c_1e^{-x/2}+c_2xe^{-x/2}$

**5.** $r^2-2=0$，$r=\pm\sqrt{2}$。$y=c_1e^{\sqrt{2}x}+c_2e^{-\sqrt{2}x}$

**6.** $r^2+9=0$，$r=\pm 3i$。$y=c_1\cos 3x + c_2\sin 3x$

**7.** $r^2-4r+5=0$，$r=2\pm i$。$y=e^{2x}(c_1\cos x + c_2\sin x)$

**8.** $r^2+r-6=0$，$(r+3)(r-2)=0$。$y=c_1e^{-3x}+c_2e^{2x}$

**9.** $9r^2+6r+1=0$，$(3r+1)^2=0$，$r=-1/3$（重根）。$y=c_1e^{-x/3}+c_2xe^{-x/3}$

**10.** $2r^2-r=0$，$r(2r-1)=0$。$y=c_1+c_2e^{x/2}$

## B 组 · 初值问题

**11.** 由 #3 通解 $y=e^{-2x}(c_1\cos 3x + c_2\sin 3x)$。$y(0)=c_1=2$。$y'(0)=-2c_1+3c_2=0$ → $c_2=4/3$。$y=e^{-2x}(2\cos 3x + \frac{4}{3}\sin 3x)$

**12.** $r^2-8r+16=0$，$r=4$（重根）。$y=c_1e^{4x}+c_2xe^{4x}$。$y(0)=c_1=4$。$y'(0)=4c_1+c_2=3$ → $c_2=-13$。$y=4e^{4x}-13xe^{4x}$

**13.** $r^2+4=0$，$r=\pm 2i$。$y=c_1\cos 2x + c_2\sin 2x$。$y(0)=c_1=1$。$y'(0)=2c_2=1$ → $c_2=1/2$。$y=\cos 2x + \frac{1}{2}\sin 2x$

**14.** $r^2-2r-3=0$，$r=3,-1$。$y=c_1e^{3x}+c_2e^{-x}$。$y(0)=c_1+c_2=2$。$y'(0)=3c_1-c_2=2$。解得 $c_1=c_2=1$。$y=e^{3x}+e^{-x}$

**15.** $r^2+4r+5=0$，$r=-2\pm i$。$y=e^{-2x}(c_1\cos x + c_2\sin x)$。$y(0)=c_1=1$。$y'(0)=-2c_1+c_2=0$ → $c_2=2$。$y=e^{-2x}(\cos x + 2\sin x)$

**16.** $r^2-4r+13=0$，$r=2\pm 3i$。$y=e^{2x}(c_1\cos 3x + c_2\sin 3x)$。$y(0)=c_1=0$。$y'(0)=2c_1+3c_2=3$ → $c_2=1$。$y=e^{2x}\sin 3x$

## C 组 · 边值问题

**17.** 通解 $y=c_1\cos 2x + c_2\sin 2x$。$y(0)=c_1=0$。$y(\pi)=c_2\sin 2\pi=0$。$y=0$（平凡解）或 $y=c_2\sin 2x$（$c_2$ 任意，全满足 $y(0)=y(\pi)=0$）。

**18.** 通解 $y=c_1\cos 3x + c_2\sin 3x$。$y(0)=c_1=1$。$y(\pi/2)=1\cdot\cos(3\pi/2)+c_2\sin(3\pi/2)=0+c_2(-1)=0$ → $c_2=0$。$y=\cos 3x$

**19.** 通解取决于 $\lambda$。$\lambda<0$ 时：不定。$\lambda=0$ 时：$y=c_1+c_2x$，$y(0)=0$ → $c_1=0$，$y(\pi/2)=c_2\pi/2=0$ → $c_2=0$，平凡解。$\lambda>0$：$y=c_1\cos(\sqrt{\lambda}x)+c_2\sin(\sqrt{\lambda}x)$。$y(0)=c_1=0$。$y(\pi/2)=c_2\sin(\sqrt{\lambda}\pi/2)=0$。有非平凡解的条件：$\sin(\sqrt{\lambda}\pi/2)=0$ → $\sqrt{\lambda}\pi/2=n\pi$ → $\sqrt{\lambda}=2n$ → $\lambda=4n^2$（$n=1,2,3,\ldots$）
