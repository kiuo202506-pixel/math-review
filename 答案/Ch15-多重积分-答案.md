# Ch15 · 多重积分 — 参考答案

## 15.1-15.2 二重积分与累次积分

**1.** $\iint_R (x+2y)dA = \int_0^2\int_1^3 (x+2y)dydx = \int_0^2 [xy+y^2]_1^3dx = \int_0^2 (3x+9-x-1)dx = \int_0^2 (2x+8)dx = [x^2+8x]_0^2 = 20$

**2.** $\int_0^1\int_0^\pi xy\cos y\,dy\,dx = \int_0^1 x\left[y\sin y+\cos y\right]_0^\pi dx = \int_0^1 x(\pi\cdot0-1-0-1)dx = \int_0^1 -2x\,dx = -1$

**3.** $\int_0^1\int_y^1 \sin(x^2)dx\,dy = \int_0^1\int_0^x \sin(x^2)dy\,dx = \int_0^1 x\sin(x^2)dx = -\frac{1}{2}\cos(x^2)\big|_0^1 = \frac{1-\cos 1}{2}$

**4.** $\int_0^2\int_{x^2}^{2x} f(x,y)dy\,dx = \int_0^4\int_{y/2}^{\sqrt{y}} f(x,y)dx\,dy$

## 15.3 一般区域上的二重积分

**5.** 区域由 $y=2x^2$（下）和 $y=1+x^2$（上）围成，$-1\le x\le 1$。$\iint = \int_{-1}^1\int_{2x^2}^{1+x^2} (x+2y)dydx = \int_{-1}^1 [xy+y^2]_{2x^2}^{1+x^2}dx = \int_{-1}^1 (x(1+x^2-2x^2)+(1+x^2)^2-(2x^2)^2)dx = \int_{-1}^1 (x-x^3+1+2x^2+x^4-4x^4)dx = \int_{-1}^1 (x-x^3+1+2x^2-3x^4)dx = 2 + \frac{4}{3} - \frac{6}{5} = \frac{32}{15}$

**6.** 交点：$y=x-1$ 与 $y^2=2x+6$ → $(x-1)^2=2x+6$ → $x^2-4x-5=0$ → $x=-1,5$。对应 $y=-2,4$。用 $y$ 做自变量更简：$x=y+1$（右边界），$x=(y^2-6)/2$（左边界）。$\iint = \int_{-2}^4\int_{(y^2-6)/2}^{y+1} xy\,dx\,dy$

**7.** 交集区域 $D$：$x^2+y^2 \le 4$（两曲面交线在 $z=4$ 平面）。体积 $V = \iint_D [(8-x^2-y^2)-(x^2+y^2)]dA = \iint_D (8-2x^2-2y^2)dA$。极坐标：$V = \int_0^{2\pi}\int_0^2 (8-2r^2)r\,dr\,d\theta = 2\pi[4r^2-\frac{r^4}{2}]_0^2 = 2\pi(16-8)=16\pi$

## 15.4 极坐标下的二重积分

**8.** $D$：第一象限，$1\le r\le 2$，$0\le\theta\le\pi/2$。$\iint = \int_0^{\pi/2}\int_1^2 e^{r^2}\cdot r\,dr\,d\theta = \frac{\pi}{2}\cdot\frac{1}{2}[e^{r^2}]_1^2 = \frac{\pi}{4}(e^4-e)$

**9.** $\iint = \int_0^\pi\int_0^2 \cos(r^2)\cdot r\,dr\,d\theta = \pi\cdot\frac{1}{2}[\sin(r^2)]_0^2 = \frac{\pi}{2}\sin 4$

**10.** $r=2\cos\theta$（内部），$r=1$（外部）。交点 $\theta=\pm\pi/3$。$A = 2\int_0^{\pi/3}\frac{1}{2}[(2\cos\theta)^2-1^2]d\theta = \int_0^{\pi/3}(4\cos^2\theta-1)d\theta = \frac{\pi}{3}+\frac{\sqrt{3}}{2}$

**11.** $r^2\le z^2$（锥面上方），$r^2+z^2\le 1$（球内）。截线：$z=\sqrt{r^2}=r$，代入球：$2r^2=1$ → $r=1/\sqrt{2}$。$V = \iint_D (\sqrt{1-r^2} - r)r\,d\theta dr = 2\pi\int_0^{1/\sqrt{2}} (r\sqrt{1-r^2}-r^2)dr = 2\pi\left[-\frac{1}{3}(1-r^2)^{3/2}-\frac{r^3}{3}\right]_0^{1/\sqrt{2}} = \frac{2\pi}{3}(1-\frac{1}{\sqrt{2}})$

## 15.6 曲面面积

**12.** $z_x=2x$，$z_y=2y$，$\sqrt{1+z_x^2+z_y^2}=\sqrt{1+4x^2+4y^2}$。$S = \iint_D \sqrt{1+4x^2+4y^2}\,dA$（$D: x^2+y^2\le 4$）。极坐标：$S = \int_0^{2\pi}\int_0^2 \sqrt{1+4r^2}\cdot r\,dr\,d\theta$。令 $u=1+4r^2$：$S = 2\pi\cdot\frac{1}{12}(17^{3/2}-1) = \frac{\pi}{6}(17^{3/2}-1)$

**13.** $z_x=3$，$z_y=4y$，$\sqrt{1+9+16y^2}=\sqrt{10+16y^2}$。$S = \int_0^1\int_0^x \sqrt{10+16y^2}\,dy\,dx$

## 15.7 三重积分（直角坐标）

**14.** $\iiint = \int_0^1\int_0^2\int_0^3 (x+y+z)dz\,dy\,dx = \int_0^1\int_0^2 [xz+yz+\frac{z^2}{2}]_0^3\,dy\,dx = \int_0^1\int_0^2 (3x+3y+\frac{9}{2})dy\,dx = \int_0^1 [3xy+\frac{3}{2}y^2+\frac{9}{2}y]_0^2dx = \int_0^1 (6x+6+9)dx = [3x^2+15x]_0^1 = 18$

**15.** $V = \int_0^1\int_0^{1-x}\int_0^{1-x-y} dz\,dy\,dx = \int_0^1\int_0^{1-x} (1-x-y)dy\,dx = \int_0^1 [(1-x)^2-\frac{(1-x)^2}{2}]dx = \int_0^1 \frac{(1-x)^2}{2}dx = \frac{1}{6}$

**16.** $E$：$x^2+y^2\le z\le 4$，$D$ 为 $x^2+y^2\le 4$。$\iiint = \iint_D\int_{x^2+y^2}^4 z\,dz\,dA = \iint_D \frac{1}{2}(16-(x^2+y^2)^2)dA$。极坐标：$\int_0^{2\pi}\int_0^2 \frac{1}{2}(16-r^4)r\,dr\,d\theta = \pi\int_0^2 (16r-r^5)dr = \pi[8r^2-\frac{r^6}{6}]_0^2 = \pi(32-\frac{64}{6}) = \frac{64\pi}{3}$

---

## 15.8 柱坐标下的三重积分

**17.** 柱坐标：圆柱外 $r\ge\sqrt{3}$，球内 $r^2+z^2\le 4$ → $-\sqrt{4-r^2}\le z\le\sqrt{4-r^2}$。$r$：$\sqrt{3}\le r\le 2$。$V = \int_0^{2\pi}\int_{\sqrt{3}}^2\int_{-\sqrt{4-r^2}}^{\sqrt{4-r^2}} r\,dz\,dr\,d\theta = 2\pi\int_{\sqrt{3}}^2 2r\sqrt{4-r^2}\,dr$。令 $u=4-r^2$：$= 2\pi\int_0^1 \sqrt{u}\,du = \frac{4\pi}{3}$

**18.** $\iiint_E y\,dV = \int_0^{\pi/2}\int_0^1\int_0^{r\cos\theta+1} (r\sin\theta)\cdot r\,dz\,dr\,d\theta = \int_0^{\pi/2}\int_0^1 r^2\sin\theta(r\cos\theta+1)dr\,d\theta$。计算得数值。

## 15.9 球坐标下的三重积分

**19.** $\iiint = \int_0^{2\pi}\int_0^\pi\int_0^4 (\rho\cos\phi)^4\cdot\rho^2\sin\phi\,d\rho\,d\phi\,d\theta = \int_0^{2\pi}d\theta\int_0^\pi \cos^4\phi\sin\phi\,d\phi\int_0^4\rho^6 d\rho = 2\pi\cdot\left[-\frac{\cos^5\phi}{5}\right]_0^\pi\cdot \frac{4^7}{7} = 2\pi\cdot\frac{2}{5}\cdot\frac{16384}{7} = \frac{65536\pi}{35}$

**20.** 球坐标：球面 $\rho=2$，锥面 $z=\sqrt{x^2+y^2}$ → $\rho\cos\phi = \rho\sin\phi$ → $\tan\phi=1$ → $\phi=\pi/4$。区域：$0\le\rho\le 2$，$0\le\phi\le\pi/4$。$V = \int_0^{2\pi}\int_0^{\pi/4}\int_0^2 \rho^2\sin\phi\,d\rho\,d\phi\,d\theta = 2\pi\cdot[-\cos\phi]_0^{\pi/4}\cdot[\frac{\rho^3}{3}]_0^2 = 2\pi\cdot(1-\frac{\sqrt{2}}{2})\cdot\frac{8}{3} = \frac{16\pi}{3}(1-\frac{\sqrt{2}}{2})$
