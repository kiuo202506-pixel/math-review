## 12.1 三维坐标系

**1.** $|PQ| = \sqrt{(4-2)^2+(2-(-1))^2+(-1-3)^2} = \sqrt{4+9+16} = \sqrt{29}$

**2.** $(x-3)^2 + (y+2)^2 + (z-5)^2 = 16$

**3.** 配方：$(x^2-4x+4)+(y^2+6y+9)+(z^2-2z+1) = 11+4+9+1$ → $(x-2)^2+(y+3)^2+(z-1)^2 = 25$。球心 $(2,-3,1)$，半径 $5$

## 12.2 向量

**4.** (a) $\langle 2, 3, 9 \rangle$ (b) $\langle 14, -19, 3 \rangle$ (c) $|\vec{a}| = \sqrt{16+4+36} = \sqrt{56} = 2\sqrt{14}$ (d) $\langle \frac{2}{\sqrt{14}}, -\frac{1}{\sqrt{14}}, \frac{3}{\sqrt{14}} \rangle$

## 12.3 点积

**7.** $\vec{a}\cdot\vec{b} = 3\cdot2 + (-1)\cdot0 + 4\cdot(-3) = 6-12 = -6$。$|\vec{a}|=\sqrt{26}$，$|\vec{b}|=\sqrt{13}$。$\cos\theta = \frac{-6}{\sqrt{26}\sqrt{13}} = \frac{-6}{\sqrt{338}}$，$\theta = \arccos(-\frac{6}{\sqrt{338}})$

**8.** $\vec{a}\cdot\vec{b} = 4+2-6 = 0$，正交。

**9.** $\langle 2,b,3 \rangle\cdot\langle -1,4,2b \rangle = -2+4b+6b = -2+10b = 0$ → $b = \frac{1}{5}$

**10.** $\text{comp}_a b = \frac{\vec{a}\cdot\vec{b}}{|\vec{a}|} = \frac{1\cdot4+2\cdot1+(-3)\cdot2}{\sqrt{14}} = \frac{4+2-6}{\sqrt{14}} = 0$

**11.** $\text{proj}_a b = \frac{\vec{a}\cdot\vec{b}}{|\vec{a}|^2}\vec{a} = \frac{0}{14}\vec{a} = \langle 0,0,0 \rangle$

**12.** $\cos\alpha = a_1/|\vec{a}|$，$\cos\beta = a_2/|\vec{a}|$，$\cos\gamma = a_3/|\vec{a}|$。$\cos^2\alpha+\cos^2\beta+\cos^2\gamma = (a_1^2+a_2^2+a_3^2)/|\vec{a}|^2 = 1$

**13.** $W = \vec{F}\cdot\vec{d} = 3\cdot4 + (-2)\cdot0 + 5\cdot(-1) = 12-5 = 7$ J

## 12.4 叉积

**14.** $\vec{a}\times\vec{b} = \begin{vmatrix}\mathbf{i}&\mathbf{j}&\mathbf{k}\\2&1&-1\\3&-2&4\end{vmatrix} = \langle 1\cdot4-(-1)(-2),\; -1\cdot3-2\cdot4,\; 2(-2)-1\cdot3 \rangle = \langle 2, -11, -7 \rangle$。验证：$\vec{a}\cdot(\vec{a}\times\vec{b}) = 4-11+7 = 0$ ✓

**15.** $|\vec{a}\times\vec{b}| = \begin{vmatrix}\mathbf{i}&\mathbf{j}&\mathbf{k}\\3&-1&2\\1&2&-4\end{vmatrix} = \langle -1(-4)-2\cdot2,\; 2\cdot1-3(-4),\; 3\cdot2-(-1)1 \rangle = \langle 0, 14, 7 \rangle$。面积 $= \sqrt{0+196+49} = \sqrt{245} = 7\sqrt{5}$

**16.** $\vec{AB} = \langle 1,3,1 \rangle$，$\vec{AC} = \langle -1,4,-1 \rangle$，$\vec{AB}\times\vec{AC} = \langle -7, 0, 7 \rangle$。面积 $= \frac{1}{2}\sqrt{49+0+49} = \frac{7\sqrt{2}}{2}$

**17.** 直线上取点 $S(1,2,3)$，$\vec{PS} = \langle -1, 1, 4 \rangle$，$\vec{v} = \langle 1, -1, 2 \rangle$。$\vec{PS}\times\vec{v} = \langle 6, 6, 0 \rangle$。$d = \frac{\sqrt{72}}{\sqrt{6}} = 2\sqrt{3}$

## 12.5 直线与平面

**18.** (a) $\vec{r} = \langle 3,-1,2 \rangle + t\langle 2,1,-4 \rangle$\
(b) $x=3+2t,\; y=-1+t,\; z=2-4t$\
(c) $\frac{x-3}{2} = \frac{y+1}{1} = \frac{z-2}{-4}$

**19.** $\vec{v} = \langle 2, 3, -5 \rangle$。参数方程：$x=1+2t,\; y=-2+3t,\; z=4-5t$

**20.** $L_1$ 方向 $\vec{v_1}=\langle 1,3,-1 \rangle$，$L_2$ 方向 $\vec{v_2}=\langle 2,1,4 \rangle$。不平行。解 $1+t=2s$，$-2+3t=3+s$，$4-t=-3+4s$。解 $s=2$，$t=3$，代入第三式：$4-3=1$，$-3+8=5$。$1\neq5$，异面。

**21.** $2(x-5)+3(y+1)-(z-3)=0$ → $2x+3y-z = 8$

**22.** $\vec{AB}=\langle 2,-3,-1 \rangle$，$\vec{AC}=\langle -1,2,-2 \rangle$，法向量 $\vec{n}=\vec{AB}\times\vec{AC} = \langle 8, 5, 1 \rangle$。平面：$8(x-1)+5(y-2)+1(z-3)=0$ → $8x+5y+z=21$

**23.** $\frac{x}{2}+\frac{y}{3}+\frac{z}{-1}=1$，即 $\frac{x}{2}+\frac{y}{3}-z=1$

**24.** $\vec{n_1}=\langle 1,2,-2 \rangle$，$\vec{n_2}=\langle 2,-1,3 \rangle$，$\cos\theta = \frac{|2-2-6|}{\sqrt{9}\cdot\sqrt{14}} = \frac{6}{3\sqrt{14}} = \frac{2}{\sqrt{14}}$，$\theta = \arccos(2/\sqrt{14})$

**25.** $d = \frac{|3\cdot2+4(-1)-12\cdot3-7|}{\sqrt{9+16+144}} = \frac{|6-4-36-7|}{13} = \frac{41}{13}$

**26.** 两平面法向量 $\vec{n_1}=\langle 1,1,-1 \rangle$，$\vec{n_2}=\langle 2,-1,3 \rangle$。交线方向 $\vec{v}=\vec{n_1}\times\vec{n_2} = \langle 2, -5, -3 \rangle$。在交线上找一点：令 $z=0$，解 $x+y=2$，$2x-y=1$ → $x=1$，$y=1$。$\vec{r} = \langle 1,1,0 \rangle + t\langle 2,-5,-3 \rangle$
