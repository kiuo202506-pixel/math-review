第15章 多重积分

15.1–15.2 二重积分 & Fubini 定理

▼叙事弧 —— 从面积求和到累次积分

一维定积分 ∫ f(x)dx 的底层是「切条 → 求和 → 取极限」。到了二维，自然想法是把平面区域切成小矩形，每个小矩形贡献 f(xij*, yij*) · ΔA，然后对 i,j 双重求和取极限。这就是二重积分的定义。

但真正算的时候没人用黎曼和。Fubini 定理告诉我们：在「好」区域上，二重积分可以拆成两次一维积分——先积 x 再积 y，或者反过来。这是本章最核心的计算技巧。

∬R f(x,y) dA = ∫ab [ ∫cd f(x,y) dy ] dx = ∫cd [ ∫ab f(x,y) dx ] dy

▼Fubini 定理（矩形区域）

若 f(x,y) 在矩形 R = [a,b] × [c,d] 上连续，则：

∬R f(x,y) dA = ∫ab ∫cd f(x,y) dy dx = ∫cd ∫ab f(x,y) dx dy

要点 次序可交换，但要确保每个积分限是常数（矩形区域下的对称性）。

▼交换积分次序（矩形区域）

当 f(x,y) = g(x)·h(y) 即被积函数可分离变量时：

∬R g(x)h(y) dA = (∫ab g(x) dx) · (∫cd h(y) dy)

直接变成两个一维积分相乘，极其方便。

易错提醒： Fubini 定理要求 f(x,y) 连续（或至少在区域上可积且累次积分存在）。遇到间断点或无界区域必须谨慎——积分次序可能影响结果甚至导致发散。

15.3 一般区域上的二重积分

▼叙事弧 —— 矩形太好算了，现实世界没有矩形区域

圆盘、三角形、由曲线围成的区域才是常态。解决思路：把一般区域描述为「夹在两条曲线之间」的条带。根据夹的方向不同分为 I 型（竖直条）和 II 型（水平条）。

▼I 型区域（x-型）

区域 D 夹在两条曲线 y = g₁(x) 和 y = g₂(x) 之间，x ∈ [a,b]：

∬D f(x,y) dA = ∫ab ∫g₁(x)g₂(x) f(x,y) dy dx

先积 y，再积 x。内层积分限是 x 的函数，外层是常数。

判断 能用竖直线贯穿区域，且交于不超过两个边界点时，适合 I 型。

▼II 型区域（y-型）

区域 D 夹在两条曲线 x = h₁(y) 和 x = h₂(y) 之间，y ∈ [c,d]：

∬D f(x,y) dA = ∫cd ∫h₁(y)h₂(y) f(x,y) dx dy

先积 x，再积 y。内层积分限是 y 的函数。

判断 能用水平线贯穿区域，且交于不超过两个边界点时，适合 II 型。

▼积分次序的选择策略

策略说明
先观察被积函数f(x,y) 对哪个变量的不定积分更简单？优先把那个变量放在内层。
再看区域形状区域是「上下夹」还是「左右夹」更自然？选择对应的类型。
考虑换序一种次序积不出来时果断换序——先画出区域，重新描述积分限。
易错提醒： 换序不是简单地把 dx dy 改成 dy dx。必须画出区域，根据图形重新写积分限。被积函数中若有无法初等积分的表达式（如 ex²），换序往往是唯一的出路。

▼自测 · 15.1–15.3

  1. 写出 ∬R xy dA（R = [0,1]×[0,2]）作为累次积分的两种形式，并算出结果。
  2. D 由 y = x² 和 y = x 围成。分别用 I 型和 II 型写出 ∬D f(x,y) dA 的累次积分表达式。
  3. 为什么 ∬D ex² dA（D 由 y = x, y = 1, x = 0 围成）必须先积 y？

15.4 极坐标下的二重积分页2

▼叙事弧 —— 「圆」的对称性呼唤极坐标

如果区域是圆、扇形、或边界含 x²+y²，直角坐标下的积分限会写得面目狰狞。极坐标将圆形区域变成矩形，代价是面积元变了。

▼坐标变换

x = r cos θ
y = r sin θ
x² + y² = r²
dA = r dr dθ
∬D f(x,y) dA = ∫αβ ∫r₁(θ)r₂(θ) f(r cos θ, r sin θ) · r dr dθ
高频失分点： 极坐标的面积元是 r dr dθ，不是 dr dθ！少乘那个 r 是本考点最经典的错误。几何解释：小扇形的面积 ≈ r·Δr·Δθ，那个 r 来自弧长。
适合极坐标的场景示例
被积函数含 x² + y²∬ (x²+y²) dA → ∬ r² · r dr dθ = ∬ r³ dr dθ
积分区域是圆/扇形/圆环圆盘 x²+y² ≤ R² → 0≤r≤R, 0≤θ≤2π
区域边界含 x²+y² 的组合圆环 1≤x²+y²≤4 → 1≤r≤2
何时用极坐标？三条判据（满足任一即可）：
  1. 被积函数中出现 x²+y² —— 直接换成 r²。
  2. 积分区域是圆、扇形、圆环 —— 直角坐标限很繁琐。
  3. 被积函数中有 x²+y² 的整体函数如 √(x²+y²)、1/(x²+y²) 等。

▼自测 · 15.4

  1. ∬D (x² + y²) dA，D 是单位圆盘 x²+y² ≤ 1。写出极坐标表达式并计算。
  2. ∬D √(x²+y²) dA，D 是圆环 1≤x²+y²≤4。答：∫₀²π ∫₁² r · r dr dθ = 2π · (8/3 − 1/3) = 14π/3。

15.6 曲面面积

▼叙事弧 —— 从平面面积到曲面面积，多一个「拉伸因子」

一元函数曲线弧长公式是 ∫ √(1+[f']²) dx。到了二元，曲面 z = f(x,y) 的面积类似地多了一个根号下的偏导数平方和——本质是切平面微元的面积相对于底面积 dA 的拉伸比。

曲面面积 S = ∬D √[ 1 + (fx)² + (fy)² ] dA

其中 D 是曲面在 xy 平面上的投影区域，fx = ∂f/∂x，fy = ∂f/∂y。

▼几何解释

曲面上一个小面片的面积 ≈ 切平面上对应平行四边形的面积。该平行四边形的边向量是 ⟨Δx, 0, fxΔx⟩ 和 ⟨0, Δy, fyΔy⟩，其叉积的模为 √[1 + (fx)² + (fy)²] Δx Δy。取极限 → 上述积分。

选择积分坐标的要点： 曲面面积的计算本质上是一个二重积分问题。选择直角坐标还是极坐标，取决于投影区域 D 的形状和被积表达式 √[1+(fx)²+(fy)²] 的复杂度。如果曲面关于 z 轴有旋转对称性，极坐标往往更合适。
易错提醒： 不要混淆曲面面积和曲面积分（第16章）。这里计算的只是 曲面的几何面积，不涉及向量场。另外，如果曲面是参数方程给出的，需要用参数形式面积公式；本章重点在显式 z = f(x,y) 的情形。

15.7 三重积分（直角坐标）

▼叙事弧 —— 再加一维，从面积到体积，从平面区域到立体

三重积分的思路和二重完全一致：把立体区域切成小长方体 → 黎曼和取极限 → Fubini 定理拆成三次一维积分。直角坐标下 dV = dx dy dz。

∭E f(x,y,z) dV = ∫ab ∫g₁(x)g₂(x) ∫u₁(x,y)u₂(x,y) f(x,y,z) dz dy dx

体积的特殊情况：当 f(x,y,z) = 1 时，三重积分就是立体区域的体积。

▼六种积分次序

三重积分有 3! = 6 种可能的积分次序（dz dy dx, dz dx dy, dy dz dx, dy dx dz, dx dy dz, dx dz dy）。选择原则：

  1. 内层：放最容易积的变量，或积分限最简洁的变量。
  2. 中层：用消去内层变量后的剩余变量之一来描述。
  3. 外层：积分限必须是常数。
积分次序决策流程：
步骤1：画出立体区域（至少想象 z 的范围）→ "底面 + 顶面" 或 "后面 + 前面"。
步骤2：确定投影区域 D（通常是 xy 平面的投影）。
步骤3：z 方向从底面 z = u₁(x,y) 到顶面 z = u₂(x,y) 先积。
步骤4：投影区域 D 按 15.3 的二重积分方式处理。
区域类型描述方式推荐次序
z 方向简单夹z₁(x,y) ≤ z ≤ z₂(x,y)先 z，再 xy
y 方向简单夹y₁(x,z) ≤ y ≤ y₂(x,z)先 y，再 xz
x 方向简单夹x₁(y,z) ≤ x ≤ x₂(y,z)先 x，再 yz
易错提醒： 三重积分换序比二重复杂得多。每种次序需要重新审视三维区域的「边界投影」——建议保持 z 最内层，然后在 xy 投影上做二重积分。

▼自测 · 15.7

  1. 用三重积分求四面体体积：x≥0, y≥0, z≥0, x+y+z≤1。
  2. 对上述区域写出 dz dy dx 形式的累次积分限。

15.8 柱坐标

▼叙事弧 —— 极坐标 + z 轴不变 = 柱坐标

柱坐标是极坐标的自然三维推广：把 (x,y) 换成 (r,θ)，z 保持不变。专门对付「旋转体」——绕 z 轴有对称性的立体。

▼坐标变换

x = r cos θ
y = r sin θ
z = z
dV = r dz dr dθ
x² + y² = r²
∭E f(x,y,z) dV = ∫αβ ∫r₁(θ)r₂(θ) ∫z₁(r,θ)z₂(r,θ) f(r cos θ, r sin θ, z) · r dz dr dθ
什么时候用柱坐标？
  1. 立体区域是绕 z 轴的旋转体（圆柱、圆锥、旋转抛物面）。
  2. 被积函数中含 x²+y² 且 z 的表达相对简单。
  3. 投影区域是圆/扇形/圆环——这往往是决定性的标志。
立体区域柱坐标描述
圆柱 x²+y² ≤ R², 0≤z≤H0≤r≤R, 0≤θ≤2π, 0≤z≤H
圆柱壳 R₁²≤x²+y²≤R₂², 0≤z≤HR₁≤r≤R₂, 0≤θ≤2π, 0≤z≤H
锥体 z = √(x²+y²), 0≤z≤Hz≤r≤H? 或 0≤r≤z≤H——需画图确定
旋转抛物面 z = x²+y², 0≤z≤H0≤r≤√H, 0≤θ≤2π, r²≤z≤H
易错提醒： 柱坐标下的体积元还是 r dz dr dθ（或 r dr dz dθ 取决于次序），和极坐标一样不要漏 r。积分次序常见 dz dr dθ（先 z）、dr dz dθ（先 r）、或 dθ 最外层不动。

▼自测 · 15.8

  1. 用柱坐标求 ∭E z dV，E 是圆柱 x²+y² ≤ 4，0 ≤ z ≤ 3。
  2. 用柱坐标求旋转抛物面 z = x²+y² 下方、平面 z = 4 上方的立体体积。

15.9 球坐标

▼叙事弧 —— 球对称时柱坐标也不够用，球坐标登场

球坐标用三个新变量描述空间点：ρ（到原点的距离）、φ（从 z 轴正方向量的角度，余纬度）、θ（在 xy 平面上的方位角，同极坐标的 θ）。适合球体、球壳、锥体等有球对称性的区域。

▼坐标变换

x = ρ sin φ cos θ
y = ρ sin φ sin θ
z = ρ cos φ

x² + y² + z² = ρ²
dV = ρ² sin φ dρ dφ dθ

范围 ρ ≥ 0，0 ≤ φ ≤ π（注意 φ 是 0 到 π，不是 2π），0 ≤ θ ≤ 2π。

高频失分点（再次强调）： 球坐标的体积元是 ρ² sin φ dρ dφ dθ。ρ² 来自径向缩放，sin φ 来自 φ 方向的弧长收缩（φ 靠近 0 或 π 时纬线变短）。不要漏掉 sin φ！而且 φ 的积分范围是 [0, π]，不是 [0, 2π]。
什么时候用球坐标？
  1. 积分区域是球体或球壳（如 x²+y²+z² ≤ R²）。
  2. 被积函数中含 x²+y²+z² 的组合（如 √(x²+y²+z²) = ρ）。
  3. 区域由一个圆锥面和一个球面围成（常见题型：冰激凌锥体）。
区域球坐标描述
球体 x²+y²+z² ≤ R²0≤ρ≤R, 0≤φ≤π, 0≤θ≤2π
上半球体（z ≥ 0 部分）0≤ρ≤R, 0≤φ≤π/2, 0≤θ≤2π
球壳 R₁²≤x²+y²+z²≤R₂²R₁≤ρ≤R₂, 0≤φ≤π, 0≤θ≤2π
冰激凌锥：锥 φ=π/4 内 + 球 ρ≤R0≤ρ≤R, 0≤φ≤π/4, 0≤θ≤2π

▼φ 的几何解释

φ = 0 对应 z 轴正方向（北极），φ = π 对应 z 轴负方向（南极），φ = π/2 对应 xy 平面（赤道）。φ 越接近 0 或 π，sin φ 越小 → 面积微元越小。这就是为什么 dV 中要乘 sin φ。

易错提醒 —— 柱坐标 vs 球坐标的选择：
不要看到 x²+y² 就条件反射用柱坐标。如果立体区域的「顶」也是曲面（如球面的一部分），球坐标可能更好。经验法则：对称轴是 z 轴 → 柱坐标；对称中心是原点 → 球坐标。

▼自测 · 15.9

  1. 用球坐标写出 ∭E (x²+y²+z²) dV，E 是单位球 x²+y²+z² ≤ 1。
  2. "冰激凌锥体"：z ≥ √(x²+y²)（即 φ ≤ π/4）的上半部分，含于球 ρ ≤ 2。写出球坐标下的积分限。
  3. 问：如果把上半球面 z = √(1−x²−y²) 写成 φ 的方程，z≥0 对应 φ 的什么范围？
