## 核心公式速查

| 概念 | 公式 |
|------|------|
| 弧长 | \(L = \int_a^b \sqrt{1+[f'(x)]^2}\,dx\) 或 \(\int_a^b \sqrt{[x'(t)]^2+[y'(t)]^2}\,dt\) |
| 旋转面面积(绕x轴) | \(S = \int_a^b 2\pi y\sqrt{1+[f'(x)]^2}\,dx\) |
| 静水压力 | \(F = \int \rho g \cdot \text{depth} \cdot \text{width}\,dy\) |
| 力矩(一维) | \(M = \sum m_k x_k\) |
| 质心(二维薄板) | \(\bar{x} = \frac{1}{A}\int x f(x)dx,\ \bar{y} = \frac{1}{A}\int \frac{1}{2}[f(x)]^2 dx\) |
| Pappus 定理 | \(V = 2\pi\bar{y}A\) (绕x轴), \(S = 2\pi\bar{y}L\) (弧绕x轴) |
