## 核心公式速查

| 概念 | 公式 |
|------|------|
| 二重积分(矩形) | ∬ f dA = ∫∫ f dx dy = ∫∫ f dy dx |
| I 型区域 | a≤x≤b, g₁(x)≤y≤g₂(x)，先积 y 后积 x |
| II 型区域 | c≤y≤d, h₁(y)≤x≤h₂(y)，先积 x 后积 y |
| 极坐标变换 | x=r cosθ, y=r sinθ, dA = r dr dθ |
| 曲面面积 | S = ∬ √[1+(f_x)²+(f_y)²] dA |
| 三重积分 | ∭ f dV（注意选择积分次序） |
| 柱坐标 | x=r cosθ, y=r sinθ, z=z, dV = r dz dr dθ |
| 球坐标 | x=ρ sinφ cosθ, y=ρ sinφ sinθ, z=ρ cosφ, dV = ρ² sinφ dρ dφ dθ |
