## 核心公式速查

| 概念 | 公式 |
|------|------|
| 点积 | a·b = a₁b₁+a₂b₂+a₃b₃ |
| 夹角 | cosθ = (a·b)/(\|a\|\|b\|) |
| 标量投影 | comp_a b = (a·b)/\|a\| |
| 向量投影 | proj_a b = [(a·b)/\|a\|²]a |
| 叉积 | a×b = det\|i j k; a₁ a₂ a₃; b₁ b₂ b₃\| |
| 平行四边形面积 | \|a×b\| |
| 平行六面体体积 | \|a·(b×c)\| |
| 直线(向量式) | r = r₀ + t v |
| 直线(对称式) | (x-x₀)/a = (y-y₀)/b = (z-z₀)/c |
| 平面(点法式) | a(x-x₀)+b(y-y₀)+c(z-z₀) = 0 |
| 点到平面距离 | d = \|ax₀+by₀+cz₀+d\|/√(a²+b²+c²) |
| 两平面夹角 | cosθ = \|n₁·n₂\|/(\|n₁\|\|n₂\|) |
