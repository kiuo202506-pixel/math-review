9.1 微分方程建模与基本概念

▼什么是微分方程？ 必考

含有未知函数及其导数的方程 = 微分方程。求解 = 找到所有满足方程的函数。

Hook：如果知道一个物体的速度，能反推出它的位置吗？能——但会多出一个任意常数（初始位置未知）。这就是微分方程的核心思想：已知变化率，反求原函数。

Why：物理定律（牛顿第二定律 F=ma = mx''(t)）、人口增长（增长率与当前人口相关）、电路（电流变化与电压关系）——这些规律天然地涉及「未知函数的变化率」，写成式子就是微分方程。

What：微分方程的阶 = 方程中出现的最高阶导数的阶。一阶：y' = f(x,y)；二阶：y'' = f(x,y,y')。一般解（通解）包含任意常数；特解由初始条件确定常数。初值问题（IVP）：给方程 + y(x₀) = y₀（一阶）或 y(x₀) = y₀, y'(x₀) = y₁（二阶）。

验证 y = Ce^(3x) 是方程 y' = 3y 的通解。

参考答案

y' = 3Ce^(3x) = 3y ✓。C 为任意常数——得一族解，每条曲线对应一个不同的初始值 y(0) = C。

▼人口增长模型 重点页1

Hook：「人口增长速度正比于当前人口」——这个假设导出什么样的增长曲线？

指数增长（Malthus 模型）：dP/dt = kP → P = P₀e^(kt)。适用于资源无限时的早期增长。

Logistic 增长：dP/dt = kP(1-P/K)。其中 K 为环境承载力（carrying capacity）。当 P ≪ K 时近似指数增长；当 P → K 时增长率 → 0。Logistic 方程是荷兰生物数学家 Verhulst 在 1840 年代提出的。

易错：指数增长和 Logistic 增长的根本区别在于「增长的刹车」——指数模型没有上限，Logistic 模型有承载力 K 做天花板。考试判断题：「人口无限增长」是对是错取决于模型假设。

9.2 方向场 — 定性分析 高频

Hook：方程 y' = f(x,y) 太难解了——能不能不求解就直接看出来解的大致形状？

方向场 (direction field) 做法：在 x-y 平面每格点画一个小线段，斜率 = f(x,y)。解曲线就是沿着这些小箭头画出的曲线。方向场特别适合判断解的渐近行为：会不会趋于某个常数（平衡解）？会不会爆炸？

欧拉方法：方向场的「数值版」——从初始点出发，沿方向场走小步得到数值近似解。yₙ₊₁ = yₙ + h·f(xₙ, yₙ)。

9.3 可分方程 必考

dy/dx = g(x)h(y) → ∫ dy/h(y) = ∫ g(x)dx

Hook：如果你的变化率 dy/dx 可以写成「x 的函数」乘「y 的函数」，你把 y 的部分挪到左边、x 的部分留在右边——两边各自积分，方程就解了。这就是「分离变量法」。

What：一阶方程 dy/dx = g(x)h(y)。1) 把 h(y) 除到左边变成 (1/h(y))·dy/dx = g(x)。2) 两边对 x 积分：∫ (1/h(y)) dy = ∫ g(x) dx。3) 解出 y（如果可以显式解出）。

典型题：dy/dx = 3x²y² → ∫ dy/y² = ∫ 3x²dx → -1/y = x³ + C → y = -1/(x³+C)。

判断：y' = xy + y² 是可分的吗？

参考答案

不是。右边 y(x+y) 不能直接分离变量，因为加号阻止了完全因式分解为 g(x)·h(y) 的乘积形式。这个方程只能用别的方法解（一阶线性方程/积分因子法）。

9.5 一阶线性方程与积分因子 必考

y' + P(x)y = Q(x) → 乘 I(x)=e^(∫Pdx) → 左边变成 (I·y)' → 积分即得 y

Hook：y' + P(x)y = Q(x) 的左边不是任何东西的导数。但如果你能「造」出一个函数 I(x)，使 I·y' + IP·y 恰好等于 (I·y)'——那左边就变成了一个乘积的导数，可以直接积分。能完成这个魔术的函数就叫积分因子。

What：要求 (I·y)' = I·y' + I'·y = I·y' + I·P·y。所以 I' = I·P → I = e^(∫Pdx)。

How（四步法）：1) 化标准形 y' + P(x)y = Q(x)；2) 求积分因子 I(x) = e^(∫Pdx)；3) 两边乘 I → (I·y)' = I·Q；4) 两边积分 → y = (∫ I·Q dx + C)/I。

易错：积分因子公式 I = e^(∫Pdx) 中指数上不要漏掉常数——不定积分时取最简单的原函数即可（常数 C 不需要写，因为 C 会出现在最后一步）。第二个常见错误：解题从不化标准形——必须先把 y' 系数归一！

解 y' + 2y = 1, y(0) = 2。

参考答案

P(x)=2, Q(x)=1。I=e^(2x)。(e^(2x)·y)' = e^(2x)。e^(2x)·y = ½e^(2x) + C。y = ½ + Ce^(-2x)。y(0)=2 → C=3/2。y = ½ + (3/2)e^(-2x)。

易错：用积分因子法时，∫P(x)dx 可能涉及需要回忆的积分公式。常见的有：∫tan x dx = -ln|cos x| + C，∫cot x dx = ln|sin x| + C，∫sec x dx = ln|sec x+tan x| + C。这些前置知识在考试中也会用到。
