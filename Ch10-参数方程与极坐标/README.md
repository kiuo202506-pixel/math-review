## 核心公式速查

| 概念 | 公式 |
|------|------|
| 参数曲线切线斜率 | \(dy/dx = (dy/dt)/(dx/dt)\) |
| 参数曲线弧长 | \(L = \int \sqrt{[x'(t)]^2 + [y'(t)]^2}\,dt\) |
| 参数曲线面积 | \(A = \int y(t)\cdot x'(t)\,dt\) |
| 极坐标面积 | \(A = \int_a^b \frac{1}{2}r^2\,d\theta\) |
| 极坐标弧长 | \(L = \int \sqrt{r^2 + (dr/d\theta)^2}\,d\theta\) |
| 极坐标切线斜率 | \(dy/dx = (r'\sin\theta + r\cos\theta)/(r'\cos\theta - r\sin\theta)\) |
