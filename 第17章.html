1 标准形与特征方程

▼二阶齐次线性常系数 ODE

a y'' + b y' + c y = 0  (a ≠ 0)

核心思路：猜测解形如 y = erx，代入得 ar²erx + br erx + c erx = 0，约去 erx ≠ 0：

ar² + br + c = 0

这就是特征方程（characteristic equation）。每个特征根 r 对应一个解 erx。通解是两个线性无关解的组合。

为什么猜 erx？ 因为常系数意味着求导不改变函数形式：(erx)' = r erx，所以 ODE 中 y, y', y'' 都含 erx 因子，代数化后只余一个关于 r 的多项式。

2 特征根的三种情形

记判别式 Δ = b² − 4ac，通解结构由 Δ 的符号决定：

Case 1 Δ > 0 — 相异实根

特征方程有两个不等实根：

r₁ ≠ r₂ ∈ ℝ

通解：

y = c₁ er₁x + c₂ er₂x

几何意义：两个指数函数的叠加，每个独立衰减或增长。

Case 2 Δ = 0 — 二重根

特征方程有唯一实根（重根）：

r = −b / (2a)

通解：

y = c₁ erx + c₂ x erx

第二个解前面多一个 x 因子（降阶法或参数变易导出）。

Case 3 Δ < 0 — 共轭复根

复根成对出现（共轭）：

r = α ± iβ

通解：

y = eαx (c₁ cos βx + c₂ sin βx)

欧拉公式 eiθ=cosθ+i sinθ 把复指数转为实三角函数。

3 三种情形速查表

判别式特征根通解解的行为特征
Δ > 0 r₁, r₂ 实数且不等 c₁er₁x + c₂er₂x 纯指数叠加，无振荡
Δ = 0 r（二重） (c₁ + c₂x) erx 临界阻尼型，多项式×指数
Δ < 0 α ± iβ eαx(c₁cosβx + c₂sinβx) 振荡 × 指数调制

4 欧拉公式在 Case 3 中的运用

▼从复指数到三角函数

当特征根为 α ± iβ 时，形式解为：

y = A e(α+iβ)x + B e(α−iβ)x

利用 欧拉公式 eiθ = cos θ + i sin θ：

y = eαx [ A eiβx + B e−iβx ] = eαx [ A(cosβx + i sinβx) + B(cosβx − i sinβx) ] = eαx [ (A+B) cosβx + i(A−B) sinβx ]

令 c₁ = A+B，c₂ = i(A−B)（均为实的任意常数），即得实通解。

记忆要点：实部 α 决定指数包络（α<0 衰减；α>0 增长；α=0 纯振荡），虚部 β 是角频率。

5 初值问题 (IVP)

▼定解两个常数

二阶方程的通解含两个任意常数 c₁, c₂，需要两个条件来确定。初值问题给出同一点的条件：

y(0) = y₀, y'(0) = y₀'

求解步骤：

  1. 写出特征方程并求根，得到通解形式。
  2. 代入 y(0) = y₀ 得第一个方程。
  3. 对通解求导得 y'(x)，代入 y'(0) = y₀' 得第二个方程。
  4. 联立解出 c₁, c₂。
易错：Case 2（重根）中通解含 xerx 项，求导时不要漏掉乘积法则！
(xerx)' = erx + r x erx = (1+rx)erx

6 边值问题 (BVP)

▼两点边界条件

边值问题在两个不同点上给定条件，例如：

y(0) = α, y(L) = β

与 IVP 的根本区别：IVP 总是存在唯一解（在合理 ODE 下），而 BVP 可能：

结果条件例子
唯一解 两边界条件与 ODE 的通解结构相容 y''+y=0, y(0)=0, y(π/2)=1
无解 边界条件自相矛盾（给定 ODE 下不可满足） y''+y=0, y(0)=0, y(π)=1
无穷多解 边界条件不足以唯一确定两常数 y''+y=0, y(0)=0, y(π)=0
为什么 BVP 会有多解 / 无解？ 因为通解中的两个常数 c₁, c₂ 由两个边界条件联立决定。若方程组的系数矩阵奇异（行列式 = 0），就会触发无解或无穷多解 —— 这正是特征值问题的入口。

7 特征值问题 (Eigenvalue Problem)

▼y'' + λ y = 0 的谱

考虑带参数 λ 的齐次方程：

y'' + λ y = 0

配以齐次边界条件（如 y(0) = y(L) = 0），寻找使非平凡解（y ≢ 0）存在的 λ 值。

分析

特征方程为 r² + λ = 0，根为 r = ±√−λ。分 λ 的符号讨论：

λ特征根通解能否满足 y(0)=y(L)=0？
λ < 0±√−λ（实）c₁ekx+c₂e−kx仅平凡解
λ = 00（二重）c₁ + c₂x仅平凡解
λ > 0±i√λc₁cos√λx + c₂sin√λx 非平凡解当 sin√λL = 0

由 y(0)=0 得 c₁=0；由 y(L)=0 得 sin(√λ L)=0 ⇒ √λ L = nπ（n = 1,2,3,…）。

特征值：

λn = (nπ / L)², n = 1,2,3,…

特征函数：

yn(x) = sin(nπx / L)
物理直觉：这是两端固定的弦的振动模态。特征值对应固有频率的平方，特征函数给出振型（sin 的节点数 = n−1）。

8 解题通用流程

  1. 标准化：将方程写为 a y'' + b y' + c y = 0。
  2. 写特征方程：a r² + b r + c = 0。
  3. 求根并分类：算 Δ = b²−4ac，判 Case 1/2/3。
  4. 写通解：按对应 case 写出含 c₁, c₂ 的通解。
  5. （如有）定常数：代入初值/边值条件，联立求 c₁, c₂。

9 易错点清单

⚠ Case 2 忘写 x 因子。重根时第二个解是 xerx 而非 erx，丢掉了线性无关性。
⚠ 复根时漏 eαx 因子。通解是 eαx(c₁cosβx+c₂sinβx)，不是纯三角函数。
⚠ 特殊方程 y''+λy=0 的符号。λ 是正数时才能振荡（sin/cos）；λ 为负时指数函数，λ=0 时直线。
⚠ 初值代入时未先求导。y'(0) 需要先对通解求导，再代 x=0，两步不可颠倒。
⚠ 边值问题直接假设唯一解。解线性方程组前先看系数行列式——为零时可能无解或无穷多解。
⚠ 复数不熟导致不敢化简。利用欧拉公式 eiθ=cosθ+i sinθ 和共轭性质，最终通解总是实的。

10 自测题

对于方程 y'' − 5y' + 6y = 0，写出通解。

点击查看答案
特征方程 r²−5r+6=0 ⇒ (r−2)(r−3)=0，r=2,3（Case 1 相异实根）。通解：y = c₁e2x + c₂e3x。

对于方程 y'' + 4y' + 4y = 0，写出通解。

点击查看答案
特征方程 r²+4r+4=0 ⇒ (r+2)²=0，r=−2（Case 2 重根）。通解：y = c₁e−2x + c₂x e−2x。

对于方程 y'' + 2y' + 5y = 0，写出通解。

点击查看答案
特征方程 r²+2r+5=0 ⇒ r=−1±2i（Case 3 共轭复根）。通解：y = e−x(c₁cos2x + c₂sin2x)。

求解初值问题：y'' + y = 0，y(0) = 1，y'(0) = 0。

点击查看答案
特征方程 r²+1=0 ⇒ r=±i（α=0, β=1），通解 y=c₁cosx+c₂sinx。
y(0)=c₁=1 ⇒ c₁=1。
y'(x)=−c₁sinx+c₂cosx，y'(0)=c₂=0 ⇒ c₂=0。
特解：y = cos x。

对 y''+λy=0, y(0)=y(π)=0，求最小的正特征值。

点击查看答案
λ>0 时通解 y=c₁cos√λx+c₂sin√λx。
y(0)=c₁=0，y(π)=c₂sin(√λπ)=0。
非平凡解要求 sin(√λπ)=0 ⇒ √λπ=nπ ⇒ λ=n²。
最小正特征值：λ=1。
