## 9.3 可分方程

$$\frac{dy}{dx} = g(x)h(y) \;\longrightarrow\; \int \frac{1}{h(y)}\,dy = \int g(x)\,dx$$

### 基础题

1. 解 $\displaystyle\frac{dy}{dx} = 3x^2y^2$。

2. 解 $\displaystyle\frac{dy}{dx} = x\sqrt{y}$。

3. 解 $xy\frac{dy}{dx} = x^2 + 1$。

4. 解 $(e^y - 1)\frac{dy}{dx} = 2 + \cos x$。

5. 解 $\displaystyle\frac{dp}{dt} = t^2p - p + t^2 - 1$。（提示：因式分解右边）

### 初值问题

6. 解 $e^{-y}\sin x - y'\cos^2 x = 0$，$y(0) = 0$。

7. 解 $\displaystyle\frac{dy}{dx} = \frac{x^2}{y(1+x^3)}$，$y(0) = 1$。

### 应用

8. 一杯咖啡温度 95°C，室温 20°C。1 分钟后咖啡降至 90°C。求任意时刻 $t$ 的温度，并计算 5 分钟后温度。（牛顿冷却定律：$\frac{dT}{dt} = -k(T - T_s)$）

9. 一个盐水槽初始含 100L 纯水。以 2L/min 的速度注入浓度 3g/L 的盐水，同时以 2L/min 的速度排出充分混合的溶液。求 $t$ 分钟后槽中盐的质量。

## 9.5 一阶线性方程

$$y' + P(x)y = Q(x),\quad I(x) = e^{\int P(x)\,dx},\quad y = \frac{1}{I(x)}\left[\int I(x)Q(x)\,dx + C\right]$$

### 基础题

10. 解 $y' + 2y = 1$，$y(0) = 2$。

11. 解 $y' - \frac{1}{2}y = 1$，$y(0) = 3$。

12. 解 $y' + \frac{2x}{1+x^2}y = 15x^2$，$y(1) = 6$。

13. 解 $y' + \frac{1}{x}y = 1$，$y(4) = 5$。

14. 解 $y' + \frac{2}{x}y = 7\sqrt{x}$，$y(1) = 9$。

15. 解 $y' + xy = x$，$y(0) = 4$。

16. 解 $y' - 2xy = 4x$，$y(0) = 1$。

17. 解 $y' - (\cot x)y = \csc x$，$y(\pi/2) = 4$。
