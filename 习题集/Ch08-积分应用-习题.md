## 8.1 弧长

$$L = \int_a^b \sqrt{1+[f'(x)]^2}\,dx \quad\text{或}\quad L = \int_a^b \sqrt{[x'(t)]^2+[y'(t)]^2}\,dt$$

### 基础题

1. 求曲线 $y = 2x - 5$，$0 \le x \le 3$ 的弧长。

2. 求曲线 $y = 1 + 6x^{3/2}$，$0 \le x \le 1$ 的弧长。

3. 求曲线 $y = \frac{2}{3}(x^2+1)^{3/2}$，$0 \le x \le 1$ 的弧长。

4. 求曲线 $y^2 = 4(x+4)^3$，$0 \le x \le 2$ 的弧长。（提示：解 $y$ 为 $x$ 的函数或用 $y$ 做自变量）

5. 求曲线 $y = \ln(\sec x)$，$0 \le x \le \pi/4$ 的弧长。

6. 求曲线 $y = \frac{1}{4}x^2 - \frac{1}{2}\ln x$，$1 \le x \le 2$ 的弧长。

### 参数曲线弧长

7. 求参数曲线 $x = t^2$，$y = 2t$，$0 \le t \le \sqrt{3}$ 的弧长。

8. 求参数曲线 $x = e^t\cos t$，$y = e^t\sin t$，$0 \le t \le \pi$ 的弧长。

### 弧长函数

9. 求曲线 $y = \frac{2}{3}x^{3/2}$ 从 $(1, \frac{2}{3})$ 开始的弧长函数 $s(x)$。

10. 已知弧长函数 $s(x) = \int_0^x \sqrt{3t+5}\,dt$，且 $f(0)=2$，求 $f(x)$。

## 8.2 旋转曲面面积

绕 $x$ 轴：$S = \int_a^b 2\pi y\sqrt{1+[f'(x)]^2}\,dx$\
绕 $y$ 轴：$S = \int_a^b 2\pi x\sqrt{1+[f'(x)]^2}\,dx$（或用 $x=g(y)$ 形式）

### 基础题

11. 设积分求曲线 $y = x^{-2}$，$1 \le x \le 2$ 绕 $x$ 轴旋转所得曲面面积（用计算器求数值）。

12. 同 11 题，绕 $y$ 轴旋转。

13. 求曲线 $y = \sin x$，$0 \le x \le \pi$ 绕 $x$ 轴旋转所得曲面的精确面积。

14. 求曲线 $y = \cos(\frac{1}{2}x)$，$0 \le x \le \pi$ 绕 $x$ 轴旋转所得曲面面积。

15. 求曲线 $y = x^3$，$0 \le x \le 1$ 绕 $x$ 轴旋转所得曲面面积。

16. 证明半径为 $r$ 的球面面积为 $4\pi r^2$。

## 8.3 物理与工程应用

### 静水压力

17. 一个水坝呈梯形：顶宽 40m，底宽 20m，高 30m。水面距坝顶 4m。求水对坝的静水总压力。

### 力矩与质心

18. 点质量系统：$m_1=6$ 在 $x=10$，$m_2=9$ 在 $x=30$。求力矩 $M$ 和质心 $\bar{x}$。

19. 求由 $y = \sqrt{x}$，$y = 0$，$x = 4$ 围成区域的形心 $(\bar{x}, \bar{y})$。

20. 求由 $y = \sin x$，$y = 0$，$0 \le x \le \pi$ 围成区域的形心。

### Pappus 定理

21. 用 Pappus 定理求半径为 $r$ 的圆绕其外部一条直线旋转所得圆环体的体积。
