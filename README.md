## 两种使用方式

| 需求 | 去哪里 |
|------|--------|
| 做练习题 | `习题集/` — 精编习题 |
| 做完对答案 | `答案/` — 逐题参考答案 |
| 搜更多在线资源 | `ChXX-*/README.md` — 外部链接+质量标签 |
