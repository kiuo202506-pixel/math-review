第10章 参数方程与极坐标

Parametric Equations and Polar Coordinates ｜ 高等数学（下）知识清单

10.1 参数方程与曲线

Hook 你扔出一颗石子。石子的轨迹既不是 x 的函数也不是 y 的函数——但它的位置 (x,y) 都是时间 t 的函数。如何描述一整条轨迹，而不必写成 y=f(x)？
TL;DR 参数方程 x=f(t), y=g(t) 用一个共同的参数 t来分别给出点的横纵坐标。曲线是由 t 的变化"驱动"画出来的轨迹。要把它变回 y=f(x) 的形式，就做消参数。

▼1. 什么是参数方程

What 曲线 C 的参数方程是一对函数：x = f(t)，y = g(t)，t 属于某个区间 I。t 每取一个值 (x,y) 就确定一个点；当 t 跑遍 I 时，(x,y) 的全体就是 C。参数 t 可以是时间、角度或任何辅助变量。

▼2. 消参数

How 策略一：直接解出 t。从 x=f(t) 中解出 t=h(x)，代入 y=g(h(x))。
策略二：利用恒等式。例如 x=cos t, y=sin t 满足 x²+y²=1，不必显式解出 t。
策略三：三角换元。形如 x=a sec t, y=b tan t 利用 1+tan² = sec² 消去。

▼3. 参数曲线作图

How 取关键 t 值（如 t=0, π/2, π 等）计算 (x,y)，标记方向（t 增大时的走向）。观察对称性：（x(t), y(t)）与（x(-t), y(-t)）的关系可判关于 x 轴 / y 轴 / 原点对称。
易错：消参数时可能丢失方向信息。例如 x=t², y=t 消参得 y²=x，但原始曲线的 t ∈ [-1,1] 只对应两个半段抛物线的一部分，不是整支抛物线！

1. 曲线 x = t² - 1, y = t + 2，消去参数 t 得 y = ?（用 x 表示）

t = y - 2，代入得 x = (y-2)² - 1 → y = 2 ± √(x+1)。注意原始 t 的范围决定取 + 还是 -。

2. 参数曲线 x = 2cos t, y = 3sin t, t∈[0,2π) 是什么曲线？方向如何？

椭圆 x²/4 + y²/9 = 1。t=0 时在 (2,0)，随 t 增大逆时针运动。

10.2 参数曲线微积分

Hook 参数曲线没有显式的 y=f(x)，那怎么求切线斜率？怎么做积分算面积和弧长？——答案是：把所有东西都写成 t 的表达式，用链式法则。
TL;DR 切线斜率 dy/dx = (dy/dt) / (dx/dt)（分母不为零时）。二阶导数继续对 x 求导：d²y/dx² = (d/dt)[dy/dx] / (dx/dt)。面积 A = ∫ y·(dx/dt) dt。弧长 L = ∫ √[(dx/dt)² + (dy/dt)²] dt。

▼1. 切线斜率

What 设 x=f(t), y=g(t) 均可导且 dx/dt ≠ 0，则 dy/dx = (dy/dt) ÷ (dx/dt) = g'(t)/f'(t)。这就是参数形式的链式法则。
Why 把 y 看作 x 的函数 y=y(x)，而 x 又是 t 的函数，于是 dy/dt = dy/dx · dx/dt，解出 dy/dx 即得上式。

▼2. 面积公式

How 曲线 C: x=f(t), y=g(t), t∈[α,β] 与 x 轴之间所围面积（y≥0 时）：
A = ∫ y dx = ∫_α^β g(t) · f'(t) dt。
关键：把 dx 换成 f'(t) dt，y 写成 g(t)，积分限换成 t 的起止值。注意积分方向——若 t:α→β 对应 x 从左到右，则无需变号。
易错：用参数面积公式时，务必检查 dx/dt 的符号！若曲线上 x 随 t 增大而减小（dx/dt < 0），直接用 ∫ y·x' dt 积分结果可能是负的——取绝对值或交换积分限。

▼3. 弧长公式

How 参数曲线 C: x=f(t), y=g(t), t∈[α,β] 的弧长为： L = ∫_α^β √[(dx/dt)² + (dy/dt)²] dt。
这个公式是"勾股定理的积分版"：一小段弧长 ds = √(dx²+dy²) = √[(x')²+(y')²] dt。

▼4. 二阶导数

What d²y/dx² = (d/dt)[dy/dx] / (dx/dt)。即先求一阶导数 dy/dx（它是 t 的函数），再对 t 求导，最后除以 dx/dt。
易错：二阶导数不是 d²y/dt² 除以 d²x/dt²！必须对 dy/dx（作为 t 的函数）再对 x 求导。

1. 曲线 x = t³, y = t²，在 t=1 处的切线斜率为？

dy/dt=2t, dx/dt=3t²，dy/dx=2/(3t)。t=1 时 dy/dx=2/3。

2. 求曲线 x = t - sin t, y = 1 - cos t (0≤t≤2π) 的弧长。

dx/dt=1-cos t, dy/dt=sin t。被积表达式 = √[(1-cos t)²+(sin t)²] = √[2-2cos t] = 2|sin(t/2)|。0≤t≤2π 时 sin(t/2)≥0，L = ∫₀²π 2sin(t/2) dt = 8。

10.3 极坐标

Hook 笛卡尔坐标系用网格线来定位，但如果图形是放射状的——花、螺旋、心形——用"距离+角度"来描述要自然得多。这就是极坐标。
TL;DR 极坐标 (r,θ) 用到原点的距离 r 和与极轴的夹角 θ 来定位点。与直角坐标的转换：x = r cos θ, y = r sin θ；反过来 r² = x² + y², tan θ = y/x（注意 θ 所在的象限）。

▼1. 极坐标定义

What 极坐标系由极点 O（原点）和极轴（正 x 轴）构成。点 P 由 (r, θ) 表示：r 是 P 到 O 的有向距离（可正可负），θ 是从极轴逆时针量起的角。r 取负值时点位于 θ+π 方向上。

▼2. 直角坐标与极坐标互化

How 极 → 直角：x = r cos θ, y = r sin θ。
直角 → 极：r = √(x²+y²)，tan θ = y/x。确定 θ 必须根据 (x,y) 所在象限调整角度。例如 (-1,1) 在第二象限，θ = 3π/4 而非 arctan(-1)= -π/4。
易错：极坐标到一个点的表示不唯一。(r,θ) 与 (r, θ+2kπ) 及 (-r, θ+(2k+1)π) 表示同一点。这在解联立方程时要特别注意。

▼3. 极坐标曲线作图

How （1）检查对称性：关于极轴（θ→-θ 不变）、关于 θ=π/2（θ→π-θ 不变）、关于极点（r→-r 或 θ→θ+π 不变）。
（2）做 r-θ 表，取关键角度（θ=kπ/6 或 kπ/4）。
（3）连点成线，标注 θ 增大时的走向。

▼4. 极坐标下的切线斜率

What 将极坐标曲线 r=f(θ) 写为参数：x=r cos θ, y=r sin θ。则 dy/dx = (dr/dθ·sin θ + r cos θ) / (dr/dθ·cos θ - r sin θ)。 水平切线：分子=0；垂直切线：分母=0。

1. 将极坐标方程 r = 2cos θ 化为直角坐标方程。

r²=2r cos θ → x²+y²=2x → (x-1)²+y²=1。这是圆心在 (1,0)、半径为 1 的圆。

2. 曲线 r=1+cos θ（心形线）关于什么对称？

cos(-θ)=cos θ，故曲线关于极轴（x 轴）对称。

10.4 极坐标下的面积与弧长页2

Hook 极坐标曲线围出的区域不是矩形条，而是扇形条。把区域切成无数个窄窄的"扇形楔子"，每个楔子的面积是 ½r²dθ——这就是极坐标积分的核心思想。
TL;DR 极坐标面积 A = ∫ ½ r² dθ = ½ ∫_α^β [f(θ)]² dθ。极坐标弧长 L = ∫_α^β √[r² + (dr/dθ)²] dθ。

▼1. 极坐标面积公式

Why 圆的面积是 πr²。一个角度为 dθ 的扇形面积占整个圆的比例为 dθ/(2π)，故 扇形微元面积 dA = ½ r² dθ。对 θ 积分即得总面积。

▼2. 两曲线之间的面积

How 设 r₁=f(θ) 和 r₂=g(θ) 交于 θ=α, β，且 f(θ)≥g(θ)≥0 在区间内，则区域面积不是像直角坐标那样 ½∫(r₁²-r₂²)dθ，而是分别积分后相减： A = ½ ∫_α^β [f(θ)² - g(θ)²] dθ。
易错：两曲线之间的面积是 ½∫(r_外² - r_内²) dθ，不是 ½∫(r_外 - r_内)² dθ。后者对应的是 r_外 - r_内 作为一条新曲线 rdθ 所得面积，完全不是同一个量。

▼3. 极坐标弧长公式

What 极坐标曲线 r=f(θ), θ∈[α,β] 的弧长为： L = ∫_α^β √[r² + (dr/dθ)²] dθ。
Why 将极坐标写为参数形式：x=r cos θ, y=r sin θ。dx/dθ = dr/dθ·cos θ - r sin θ，dy/dθ = dr/dθ·sin θ + r cos θ。计算 (dx/dθ)²+(dy/dθ)²，化简即得 r²+(dr/dθ)²。

▼4. 常见极坐标曲线的面积

曲线方程一瓣面积
圆r = aπa²（一周）
心形线r = a(1+cos θ)3πa²/2
三叶玫瑰线r = a cos(3θ)πa²/4（一瓣）
双纽线r² = a² cos(2θ)a²/2（一瓣）
Checkpoint 本章最重要的两个公式——极坐标面积和弧长——一旦混淆将导致完全错误的答案。面积是 ½∫r²dθ（平方后在积分号内），弧长是 ∫√(r²+r'²)dθ（被积函数有根号）。牢记这一点。

1. 求 r = 1 + cos θ（心形线）所围总面积。

A = ½∫₀²π (1+cos θ)² dθ = ½∫(1+2cos θ+cos²θ)dθ。利用 cos²θ=(1+cos 2θ)/2，得 A = ½[θ+2sin θ+½θ+¼sin 2θ]₀²π = ½(3π) = 3π/2。

2. 求 r = e^θ 从 θ=0 到 θ=2π 的弧长。

r' = e^θ。L = ∫₀²π √(e²θ+e²θ) dθ = ∫₀²π √2·e^θ dθ = √2·(e²π-1)。

综合自测

1. 把直角坐标方程 x³ = y² 改为参数方程形式。

可令 x=t², y=t³，则 x³ = t⁶ = y²。或令 y=t, x=t^(2/3)。

2. 参数曲线 x=√t, y=1-t (t≥0) 的切线何时水平？

dy/dt=-1, dx/dt=1/(2√t)，dy/dx=-2√t。水平切线需 dy/dx=0 → t=0，此时点 (0,1)。

3. 极坐标曲线 r=sin(2θ) 有几瓣？一瓣面积是多少？

四叶玫瑰线，共 4 瓣。一瓣对应 θ∈[0,π/2]，A=½∫₀^{π/2} sin²(2θ) dθ = π/8。
