第12章 向量与空间几何

12.1 三维坐标系 · 12.2 向量 · 12.3 点积 · 12.4 叉积 · 12.5 空间直线与平面

三句话叙事弧：从数轴到空间 —— 定位点；从点到向量 —— 刻画方向与大小； 从向量运算到几何对象 —— 直线、平面、距离，一以贯之。

12.1 三维坐标系

三维直角坐标系：从原点 O 出发，三条互相垂直的轴：x 轴、y 轴、z 轴，满足右手定则。

▼两点距离

|P₁P₂| = √[(x₂−x₁)² + (y₂−y₁)² + (z₂−z₁)²]

本质是勾股定理在三维的直接推广。

▼球面方程

以 C(h, k, l) 为球心、半径为 r：

(x − h)² + (y − k)² + (z − l)² = r²

展开后形如 x²+y²+z² + Ax+By+Cz+D=0，配方后半径²为正才是真球面。

12.2 向量

向量 = 大小 + 方向。记作 a 或 AB⃗。零向量 0 长度为 0，方向不定。

▼基本运算

运算定义几何意义
加法 a+b分量相加：⟨a₁+b₁, a₂+b₂, a₃+b₃⟩平行四边形法则 / 三角形法则
减法 a−b分量相减a+(−b)，差向量从 b 末端指向 a 末端
数乘 ca每分量乘 c缩放：|c| 倍长，c<0 反向

▼分量、模、单位向量

向量 a = ⟨a₁, a₂, a₃⟩ → 标准基展开：a = a₁i + a₂j + a₃k

|a| = √(a₁² + a₂² + a₃²)
u = a / |a|　（单位向量，方向同 a，长度为 1）
区分"向量"与"标量"：向量有方向。长度 / 面积 / 体积是标量，位移 / 速度 / 力是向量。

12.3 点积 (Dot Product)

点积（数量积、内积）：a·b = |a||b| cos θ

▼分量公式

a·b = a₁b₁ + a₂b₂ + a₃b₃

▼核心推论

用途公式
夹角cos θ = a·b / (|a||b|)
正交判定a·b = 0 ⇔ a ⟂ b
标量投影compa b = a·b / |a|
向量投影proja b = (a·b / |a|²) a
方向余弦cos α = a₁/|a|, cos β = a₂/|a|, cos γ = a₃/|a|;　cos²α+cos²β+cos²γ = 1

▼物理应用：功

W = F·d = |F||d| cos θ

只有沿位移方向的分力才做功。

▼运算律

a·b = b·a（交换律）  a·(b+c) = a·b + a·c（分配律）  (ca)·b = c(a·b)

投影 compa b 是标量，可正可负；proja b 是向量，与 a 同向或反向。切勿混淆两者。

12.4 叉积 (Cross Product)

叉积（向量积、外积）：a×b 是一个与 a、b 都正交的向量，方向由右手定则确定。| a×b | = |a||b| sin θ。

▼行列式公式

| i j k |
a×b = | a₁ a₂ a₃ |
| b₁ b₂ b₃ |

= ⟨ a₂b₃−a₃b₂ , a₃b₁−a₁b₃ , a₁b₂−a₂b₁ ⟩

▼几何与物理意义

量公式说明
平行四边形面积Area = |a×b|三角形面积 = ½|a×b|
平行六面体体积V = |a·(b×c)|三向量为棱，取其绝对值
扭矩τ = r×Fr 为力臂，F 为力

▼正交性验证

叉积结果与两个原向量都正交：(a×b)·a = 0 且 (a×b)·b = 0

▼运算律

a×b = −(b×a)　（反交换律 — 这一点与点积截然不同）

(ca)×b = c(a×b)　 　a×(b+c) = a×b + a×c

叉积不满足交换律：a×b = −b×a。右手定则记法：食指 a → 中指 b → 拇指方向即 a×b。

点积 vs 叉积 对比

点积 a·b叉积 a×b
结果类型标量向量
几何含义投影长度 × 模；衡量"同向度"平行四边形的有向面积；衡量"正交度"
交换律✓ 满足✗ 反交换：a×b = −b×a
平行时|a||b| (cos 0=1)0 (sin 0=0)
正交时0 (cos 90°=0)|a||b| (sin 90°=1)
典型应用夹角、投影、功面积、体积、扭矩、法向量

12.5 空间直线与平面

▼直线 三种表示形式

过点 P₀(x₀,y₀,z₀)，方向向量 v = ⟨a, b, c⟩。

形式表达式
向量式r = r₀ + tv　(t ∈ ℝ)
参数式x=x₀+at, y=y₀+bt, z=z₀+ct
对称式(x−x₀)/a = (y−y₀)/b = (z−z₀)/c

若某个方向分量为 0（如 a=0），则对称式中该项写为 x=x₀，参数式中该坐标恒为常数。

▼直线间关系

平行：方向向量成比例　v₁ ∥ v₂

垂直：方向向量点积为 0　v₁·v₂ = 0

共面：(r₂−r₁)·(v₁×v₂) = 0

异面：(r₂−r₁)·(v₁×v₂) ≠ 0

异面时最短距离：d = |(r₂−r₁)·(v₁×v₂)| / |v₁×v₂|

▼平面 四种表示形式

形式表达式说明
点法式A(x−x₀) + B(y−y₀) + C(z−z₀) = 0过 P₀，法向量 n = ⟨A,B,C⟩
一般式Ax + By + Cz + D = 0法向量即 ⟨A,B,C⟩
截距式x/p + y/q + z/r = 1截距 p,q,r 均 ≠ 0

▼平面间夹角

即两法向量的夹角（取锐角）：

cos θ = |n₁·n₂| / (|n₁||n₂|)

▼点到平面距离页5

d = |Ax₀ + By₀ + Cz₀ + D| / √(A² + B² + C²)

点 P₀(x₀,y₀,z₀) 到平面 Ax+By+Cz+D=0 的距离。

▼直线与平面的关系

关系条件
直线 ⟂ 平面方向向量 v 与法向量 n 平行：v ∥ n
直线 ∥ 平面v·n = 0 且直线上某点不在平面内
直线在平面内v·n = 0 且直线上一点满足平面方程
直线与平面斜交v·n ≠ 0

直线与平面夹角（锐角）：sin φ = |v·n| / (|v||n|)

平面一般式 Ax+By+Cz+D=0 中，⟨A,B,C⟩ 是法向量，不是方向向量。直线对称式中的 a,b,c 是方向向量的分量。方向向量与法向量是两种完全不同的角色，切勿张冠李戴。
对称式中若某分母为 0，不可直接写分式。例如 a=0 时应理解为 x=x₀且 (y−y₀)/b = (z−z₀)/c。考试中若机械地"除以零"会丢分。

几何对象快速对照

对象决定要素关键向量核心公式
直线一点 + 一方向方向向量 vr=r₀+tv
平面一点 + 一法向法向量 nn·(r−r₀)=0
球面球心 + 半径—(x−h)²+(y−k)²+(z−l)²=r²
